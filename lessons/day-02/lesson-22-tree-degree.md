# 第八章 8.2 图的基本概念：树的度数

在图论中，树是连通且无圈的无向图。

## 1. 顶点的度

树中某个顶点的度，就是与该顶点直接相连的边数。

记作：

```text
d(v)
```

例如某个顶点连接了3条边，则：

```text
d(v)=3
```

## 2. 叶子点

在树中，度为1的顶点叫叶子点，也可看作悬挂点。

```text
d(v)=1
```

## 3. 树的度

有时说“树的度”，通常指这棵树中所有顶点度数的最大值。

```text
树的度 = max d(v)
```

例如各顶点度数为：

```text
1, 3, 2, 1, 1
```

则这棵树的度为：

```text
3
```

## 4. 度数和公式

如果一棵树有 `n` 个顶点，则边数为：

```text
n-1
```

所有顶点度数之和等于边数的2倍：

```text
sum d(v) = 2(n-1)
```

这是因为每条边连接两个端点，对度数总和贡献2。

## 5. 根树中的度

若把树画成有根树，根在上、子节点在下，则有时“度”也指某个结点的孩子个数。

注意区分：

- 图论中的顶点度：与该顶点相连的边数；
- 有根树中的结点度：该结点的孩子个数。

在运筹学图论中，优先按图论理解：度数等于关联边数。
