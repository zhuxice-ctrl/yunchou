# 第八章 8.2 图的基本概念

## 1. 图

图通常记为：

```text
G=(V,E)
```

其中：

- `V` 是顶点集合；
- `E` 是边集合。

顶点可以表示地点、事件、工序、城市、仓库等对象；边表示两个对象之间存在某种联系。

## 2. 顶点

顶点也叫点，通常用：

```text
v1, v2, ..., vn
```

表示。

顶点集合记为：

```text
V={v1,v2,...,vn}
```

## 3. 边

无向图中的边通常记为：

```text
[vi,vj]
```

表示顶点 `vi` 与顶点 `vj` 之间有一条无方向的边。

有向图中的弧通常记为：

```text
(vi,vj)
```

表示从 `vi` 指向 `vj`。

## 4. 无向图

图中每条边没有方向，称为无向图。

例如：

```text
[v1,v2]
```

只表示 `v1` 与 `v2` 相连，不区分方向。

## 5. 有向图

图中每条边有方向，称为有向图。带方向的边一般叫弧。

例如：

```text
(v1,v2)
```

表示从 `v1` 到 `v2`，方向不能随意反过来。

## 6. 端点

一条边连接的两个顶点叫这条边的端点。

例如边 `[v1,v2]` 的端点是：

```text
v1 和 v2
```

## 7. 关联

如果一条边以某个顶点为端点，就说这条边与该顶点关联。

例如 `[v1,v2]` 与 `v1` 关联，也与 `v2` 关联。

## 8. 邻接

如果两个顶点之间有边直接相连，就称这两个顶点邻接。

例如存在边 `[v1,v2]`，则 `v1` 与 `v2` 邻接。

如果两条边有公共端点，也可以说这两条边相邻。

## 9. 次或度

在无向图中，与顶点 `vi` 关联的边数称为该顶点的次，也叫度，记为：

```text
d(vi)
```

如果一个顶点连接了3条边，则：

```text
d(vi)=3
```

## 10. 入次和出次

在有向图中：

- 进入顶点的弧数叫入次；
- 从顶点发出的弧数叫出次。

通常：

```text
d-(vi): 入次
d+(vi): 出次
```

总度数为：

```text
d(vi)=d-(vi)+d+(vi)
```

## 11. 孤立点

若某个顶点没有任何边与它关联，则称为孤立点。

也就是：

```text
d(vi)=0
```

## 12. 悬挂点与悬挂边

若某个顶点的度为1，则称该顶点为悬挂点。

与悬挂点关联的那条边称为悬挂边。

## 13. 环

一条边的两个端点相同，称为环。

例如：

```text
[v1,v1]
```

## 14. 平行边或重边

两个顶点之间有两条或两条以上的边，称为平行边或重边。

## 15. 简单图

没有环，也没有平行边的图，称为简单图。

## 16. 完全图

在无向简单图中，任意两个不同顶点之间都有一条边相连，称为完全图。

n个顶点的完全图常记为：

```text
Kn
```

## 17. 链

在无向图中，顶点和边交替出现的序列称为链。

形式如：

```text
v0, e1, v1, e2, v2, ..., ek, vk
```

若边的具体名称不重要，也可以简写为顶点序列：

```text
v0, v1, v2, ..., vk
```

## 18. 路

如果链中各条边互不相同，称为路。

若各顶点也互不相同，则通常称为初等路。

## 19. 回路

如果一条链的起点和终点相同，称为回路。

即：

```text
v0=vk
```

## 20. 圈

起点和终点相同，且中间顶点不重复的回路，称为圈。

## 21. 连通

若两个顶点之间存在一条链，则称这两个顶点连通。

## 22. 连通图

如果无向图中任意两个顶点都是连通的，则该图称为连通图。

若不是任意两点都连通，则称为非连通图。

## 23. 连通分量

非连通图可以分成若干个互不连通的部分，每个最大的连通子图称为一个连通分量。

## 24. 子图

若一个图的顶点和边都分别取自另一个图，则称它是原图的子图。

## 25. 生成子图

若子图包含原图的全部顶点，只删去部分边，则称为生成子图。

## 26. 树

连通且无圈的无向图称为树。

树是第八章后续最小生成树问题的基础概念。

## 27. 生成树

连通图的一个生成子图如果是一棵树，则称为该图的生成树。

生成树包含原图的全部顶点，但只保留使图连通且不形成圈的一部分边。

## 28. 权

边或弧上给出的数量称为权。

权可以表示：

- 距离；
- 时间；
- 费用；
- 容量；
- 收益。

带权的图称为赋权图或网络。

## 29. 网络

在图中给边或弧赋予权数后，称为网络。

例如最短路问题中，边权通常表示距离；最大流问题中，弧权通常表示容量。

## 30. 本节考试记忆主线

```text
图 = 顶点 + 边
边连接顶点
有方向的是有向图，没有方向的是无向图
边数对应顶点的度
能走到叫连通
起终点相同叫回路或圈
连通且无圈叫树
带权图叫网络
```
