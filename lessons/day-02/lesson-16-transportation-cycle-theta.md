# 第四章 运输问题：闭回路调整量 theta

当某个空格检验数为正时，选该空格入基。

本例中：

```text
r24 = 1 > 0
```

所以选 `(2,4)` 入基。

构造闭回路：

```text
(2,4) -> (2,3) -> (1,3) -> (1,4) -> (2,4)
```

从入基空格开始标 `+`，沿闭回路交替标号：

```text
+ (2,4)
- (2,3)
+ (1,3)
- (1,4)
```

调整量 theta 不是取所有标记格的值，而是只取负号格中已有运输量的最小值：

```text
theta = min{x23, x14}
      = min{1, 3}
      = 1
```

这样做是为了保证减号格减去 theta 后不会出现负运输量。

调整规则：

```text
正号格：加 theta
负号格：减 theta
其他格：不变
```

本例：

```text
x24 = 0 + 1 = 1
x23 = 1 - 1 = 0
x13 = 4 + 1 = 5
x14 = 3 - 1 = 2
```

如果 `(2,3)` 的运输量是2，而 `(1,4)` 仍是3，则：

```text
theta = min{2,3}=2
```

如果某个负号格为0，则 theta=0，属于退化情形。

负号格不应该出现负运输量。运输方案要求 `xij>=0`，若负号格原本就是负数，说明当前运输方案不合法；若调整后出现负数，说明 theta 取大了。
