# 第八章 最小树问题：权如何计算

在最小权支撑树问题中，图上每条边旁边标出的数字就是这条边的权。

权可以表示：

```text
距离、费用、时间、长度、造价
```

具体含义由题目背景决定。

## 1. 边的权

例如边 `CF` 旁边标着 `1`，则：

```text
w(CF)=1
```

边 `AC` 旁边标着 `2`，则：

```text
w(AC)=2
```

## 2. 树的权

一棵支撑树的权，等于这棵树中所有选中边的权数之和。

```text
树的权 = 选中边的权数相加
```

不是加顶点数，也不是加度数。

## 3. 本图一种最小权支撑树

选边：

```text
CF, AC, DF, EF, AB, DG
```

对应权数：

```text
1, 2, 2, 2, 3, 4
```

所以总权为：

```text
1+2+2+2+3+4=14
```

因此这棵支撑树的权为：

```text
14
```

## 4. 记忆

```text
边上的数字 = 边的权
选中的边权相加 = 这棵树的权
所有支撑树中权最小的 = 最小权支撑树
```
