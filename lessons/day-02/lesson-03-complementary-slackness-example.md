# 第二章 第3课：松紧互补例题

## 课件中的原问题

```text
max z = 2x1 + 4x2 + 3x3

s.t.
    3x1 + 4x2 + 2x3 <= 60
    2x1 + x2 + 2x3 <= 40
    x1 + 3x2 + 2x3 <= 50
    x1, x2, x3 >= 0
```

对偶问题：

```text
min f = 60y1 + 40y2 + 50y3

s.t.
    3y1 + 2y2 + y3 >= 2
    4y1 + y2 + 3y3 >= 4
    2y1 + 2y2 + 2y3 >= 3
    y1, y2, y3 >= 0
```

## 松紧互补的两组对应

### 原变量与对偶约束

```text
xj > 0  => 第j条对偶约束取等号
```

课件给出：

```text
x = (0, 20/3, 50/3)^T
```

因为：

```text
x2 > 0
x3 > 0
```

所以对偶第2、3条约束取等号：

```text
4y1 + y2 + 3y3 = 4
2y1 + 2y2 + 2y3 = 3
```

`x1=0` 时，对偶第1条约束不必取等号，只要求保持可行。

### 原约束与对偶变量

```text
原问题第i条约束有严格松弛 => yi = 0
```

课件按第三条原约束有松弛处理，因此写：

```text
y3 = 0
```

代入前两个等式：

```text
4y1 + y2 = 4
2y1 + 2y2 = 3
```

解得：

```text
y1 = 5/6
y2 = 2/3
y3 = 0
```

## 课件数据核对提醒

按截图中给出的 `x=(0,20/3,50/3)^T` 代入第三条原约束：

```text
x1 + 3x2 + 2x3 = 160/3
```

而 `160/3 > 50`，与课件中标注的 `< 50` 不一致。因此该页课件很可能在 `x` 的数值、第三条约束右端或不等号标注中存在笔误。理解松紧互补时应以对应规则为主，但做具体计算时必须先核对可行性。
