# 第五章 指派问题：圈零元素法

课件矩阵：

```text
1 3 0 11 8
0 0 6  6 2
0 1 2  1 0
1 0 5  0 4
1 2 3  4 0
```

独立零元素：选出的零元素之间不能在同一行，也不能在同一列。

圈零元素法：

1. 先找只有一个零元素的行或列，将该零元素圈起来。
2. 圈住一个零元素后，划去同一行和同一列的其他零元素。
3. 重复，直到不能继续。
4. 若圈出的独立零元素个数等于矩阵阶数 n，则得到一个完整指派方案。

本例：

- 第1行只有第3列为0，所以圈 `(1,3)`。
- 第5行只有第5列为0，所以圈 `(5,5)`。
- 第3行原有第1列和第5列为0，但第5列已被占用，所以圈 `(3,1)`。
- 第2行第1列、第2列为0，但第1列已被占用，所以圈 `(2,2)`。
- 第4行第2列、第4列为0，但第2列已被占用，所以圈 `(4,4)`。

独立零元素集合为：

```text
(1,3), (2,2), (3,1), (4,4), (5,5)
```

对应指派方案：

```text
1 -> 3
2 -> 2
3 -> 1
4 -> 4
5 -> 5
```

因为一共圈出5个独立零元素，等于矩阵阶数5，所以已经得到完整指派。
