# 第二章 第1课：2.1 对偶问题

> 本讲严格按照教师课件《2.1 对偶问题》的结构学习：原问题、对偶问题、对偶问题的写法、对称型与非对称型。

## 一、一个线性规划问题对应两个问题

课件把原来的线性规划称为：

```text
原问题（primal problem，P）
```

与它对应的另一个线性规划称为：

```text
对偶问题（dual problem，D）
```

原问题与对偶问题是一一对应的。

## 二、先学习课件中的对称型

设原问题为：

```text
max z = c1x1 + c2x2 + ... + cnxn

s.t.
    a11x1 + a12x2 + ... + a1nxn <= b1
    a21x1 + a22x2 + ... + a2nxn <= b2
    ...
    am1x1 + am2x2 + ... + amnxn <= bm
    x1, x2, ..., xn >= 0
```

它的对偶问题写为：

```text
min w = b1y1 + b2y2 + ... + bmym

s.t.
    a11y1 + a21y2 + ... + am1ym >= c1
    a12y1 + a22y2 + ... + am2ym >= c2
    ...
    a1ny1 + a2ny2 + ... + amnym >= cn
    y1, y2, ..., ym >= 0
```

## 三、课件写法的对应关系

### 1. max 与 min 对换

```text
原问题 max  ↔  对偶问题 min
```

### 2. 原问题约束个数 = 对偶变量个数

若原问题有 `m` 条约束，则对偶问题有 `m` 个变量：

```text
y1, y2, ..., ym
```

### 3. 原问题变量个数 = 对偶约束个数

若原问题有 `n` 个变量，则对偶问题有 `n` 条约束。

### 4. 原问题右端常数变成对偶目标函数系数

```text
b1, b2, ..., bm
```

变成：

```text
min w = b1y1 + b2y2 + ... + bmym
```

### 5. 原问题目标函数系数变成对偶约束右端常数

```text
c1, c2, ..., cn
```

分别放到对偶问题各条约束右端。

### 6. 系数矩阵转置

原问题系数矩阵按“行”排列；写对偶时按“列”读取。

```text
A  →  A^T
```

## 四、完整例题

原问题：

```text
max z = 3x1 + 2x2

s.t.
    x1 + 2x2 <= 4
    2x1 + x2 <= 5
    x1, x2 >= 0
```

### 第一步：确定对偶变量

原问题有两条约束，因此对偶问题有两个变量：

```text
y1, y2 >= 0
```

### 第二步：写对偶目标函数

原问题右端常数是4和5，因此：

```text
min w = 4y1 + 5y2
```

### 第三步：按原问题变量列写对偶约束

原问题中 `x1` 列为：

```text
1, 2
```

目标函数中 `x1` 的系数为3，所以第一条对偶约束：

```text
y1 + 2y2 >= 3
```

原问题中 `x2` 列为：

```text
2, 1
```

目标函数中 `x2` 的系数为2，所以第二条对偶约束：

```text
2y1 + y2 >= 2
```

### 最终对偶问题

```text
min w = 4y1 + 5y2

s.t.
    y1 + 2y2 >= 3
    2y1 + y2 >= 2
    y1, y2 >= 0
```

## 五、按课件记忆

```text
原问题约束 → 对偶变量
原问题变量 → 对偶约束
右端常数 b → 对偶目标系数
目标系数 c → 对偶约束右端
系数矩阵 A → 转置 A^T
max、<=、x>=0 → min、>=、y>=0
```

## 六、常见错误

1. 对偶变量个数写成原问题变量个数。
2. 没有按列读取原问题系数矩阵。
3. 把原问题右端常数与目标系数的位置弄反。
4. 忘记将 `max` 改成 `min`。
5. 忘记将对偶约束写成 `>=`。
6. 漏写对偶变量非负约束。
