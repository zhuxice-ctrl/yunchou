# 第三章 3.4 分枝定界法：松弛问题的 Greedy 求解

课件例题：

```text
max z = 4x1 + 3x2 + 2x3
s.t.
    2x1 + 5x2 + 3x3 <= 7
    x1,x2,x3 ∈ {0,1}
```

将 0-1 条件松弛为：

```text
0 <= x1,x2,x3 <= 1
```

得到单约束连续背包问题：

```text
max z = 4x1 + 3x2 + 2x3
s.t.
    2x1 + 5x2 + 3x3 <= 7
    0 <= x1,x2,x3 <= 1
```

按单位资源收益 `ci/ai` 排序：

```text
x1: c1/a1 = 4/2 = 2
x3: c3/a3 = 2/3
x2: c2/a2 = 3/5
```

所以顺序为：

```text
x1 -> x3 -> x2
```

依次装入：

1. 令 `x1=1`，占用资源 2，剩余资源 `7-2=5`。
2. 令 `x3=1`，再占用资源 3，剩余资源 `5-3=2`。
3. `x2` 完全取 1 需要资源 5，但只剩 2，所以取：

```text
x2 = 2/5
```

得到松弛问题最优解：

```text
x=(1,2/5,1)^T
```

目标值：

```text
z = 4*1 + 3*(2/5) + 2*1
  = 36/5
  = 7.2
```

该解不是整数解，因为 `x2=2/5`。但对于原最大化整数规划，松弛问题最优值 `36/5` 是一个上界。

课件中的 Greedy 规则只适用于这种“单个容量约束、变量范围为 0 到 1”的连续背包松弛问题，不能直接推广到任意线性规划。
