# 第三章 3.3 割平面法例题（按课件）

原整数规划：

```text
max Z = x1 + x2
s.t.
    3x1 + x2 <= 4
   -x1 + x2 <= 1
    x1, x2 >= 0，且为整数
```

课件将目标函数改写为：

```text
min z = -x1 - x2
```

加入松弛变量后：

```text
3x1 + x2 + x3 = 4
-x1 + x2 + x4 = 1
x1,x2,x3,x4 >= 0
```

松弛问题的最终最优单纯形表为：

```text
x_B | x1  x2   x3    x4  | b̄
x1  | 1   0    1/4  -1/4 | 3/4
x2  | 0   1    1/4   3/4 | 7/4
z   | 0   0   -1/2  -1/2 | -5/2
```

所以松弛问题最优解为：

```text
x1=3/4, x2=7/4
```

原最大化目标值为 `5/2`，但不是整数解。

右端小数部分：

```text
f10 = 3/4
f20 = 3/4
```

课件在并列时选择第一行作为源行，因此 `f10=3/4`。

源行为：

```text
x1 + (1/4)x3 - (1/4)x4 = 3/4
```

小数部分定义为：

```text
f = a - floor(a)
```

所以：

```text
f13 = 1/4
f14 = -1/4 - floor(-1/4)
    = -1/4 - (-1)
    = 3/4
```

割平面：

```text
f10 - (f13 x3 + f14 x4) <= 0
```

代入：

```text
3/4 - (1/4 x3 + 3/4 x4) <= 0
```

等价于：

```text
-1/4 x3 - 3/4 x4 <= -3/4
```

引入松弛变量 `x5>=0`：

```text
-1/4 x3 - 3/4 x4 + x5 = -3/4
```

加入原最优表：

```text
x_B | x1  x2   x3    x4   x5 | b̄
x1  | 1   0    1/4  -1/4   0 | 3/4
x2  | 0   1    1/4   3/4   0 | 7/4
x5  | 0   0   -1/4  -3/4   1 | -3/4
z   | 0   0   -1/2  -1/2   0 | -5/2
```

右端出现负数，因此使用对偶单纯形法。离基变量为 `x5`。

对候选入基列计算：

```text
x3: (-1/2)/(-1/4) = 2
x4: (-1/2)/(-3/4) = 2/3
```

按课件选较小者，因此 `x4` 入基，枢轴元素为 `-3/4`。

将枢轴行除以 `-3/4`：

```text
R3' = R3 / (-3/4)
```

得到：

```text
x4 + 1/3 x3 - 4/3 x5 = 1
```

消去其他行的 `x4`：

```text
R1' = R1 + 1/4 R3'
R2' = R2 - 3/4 R3'
Rz' = Rz + 1/2 R3'
```

最终表：

```text
x_B | x1  x2   x3  x4    x5 | b̄
x1  | 1   0    1/3  0   -1/3| 1
x2  | 0   1    0    0    1  | 1
x4  | 0   0    1/3  1   -4/3| 1
z   | 0   0   -1/3  0   -2/3| -2
```

令非基变量 `x3=x5=0`，得到：

```text
x1=1, x2=1
```

均为整数，因此整数规划最优解为：

```text
x*=(1,1)
Zmax=2
```

本题只加入一次割平面就得到整数最优解。
