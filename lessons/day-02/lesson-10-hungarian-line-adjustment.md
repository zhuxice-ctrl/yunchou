# 第五章 指派问题：匈牙利法画线加减过程

以排课表例题的约化矩阵为例：

```text
        微积分  线性代数  概率论  运筹学
A         0       8       2      5
B        11       0       5      4
C         2       3       0      0
D         0      11       4      5
```

零元素位置：

```text
A: 微积分
B: 线性代数
C: 概率论、运筹学
D: 微积分
```

此时不能找到4个独立零元素，需要用最少直线覆盖所有零元素。

一种最少画线方法：

```text
第1列、B行、C行
```

覆盖关系：

- 第1列覆盖 `(A,微积分)` 和 `(D,微积分)`；
- B行覆盖 `(B,线性代数)`；
- C行覆盖 `(C,概率论)` 和 `(C,运筹学)`。

未被覆盖的元素是没有横线、也没有竖线的位置：

```text
A行: 线性代数8、概率论2、运筹学5
D行: 线性代数11、概率论4、运筹学5
```

其中最小值：

```text
theta = 2
```

课件调整规则：

```text
未被直线覆盖的元素：减 theta
两条直线交叉处的元素：加 theta
只被一条直线覆盖的元素：不变
```

逐项调整：

```text
A行: 0不变, 8-2=6, 2-2=0, 5-2=3
B行: 11+2=13, 0不变, 5不变, 4不变
C行: 2+2=4, 3不变, 0不变, 0不变
D行: 0不变, 11-2=9, 4-2=2, 5-2=3
```

得到新矩阵：

```text
        微积分  线性代数  概率论  运筹学
A         0       6       0      3
B        13       0       5      4
C         4       3       0      0
D         0       9       2      3
```

新增零元素在 `(A,概率论)`，于是可以圈出4个独立零元素：

```text
A -> 概率论
B -> 线性代数
C -> 运筹学
D -> 微积分
```

回到原始费用矩阵计算总费用：

```text
A -> 概率论: 9
B -> 线性代数: 4
C -> 运筹学: 11
D -> 微积分: 4
总费用 = 28
```

核心记忆：

```text
没盖住的减 theta，交叉处加 theta，一条线盖住的不动。
```
