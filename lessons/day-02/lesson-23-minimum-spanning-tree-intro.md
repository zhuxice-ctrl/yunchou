# 第八章：最小树问题与最小权支撑树

课件概念：

```text
最小树问题 = minimum weight spanning tree
```

## 1. 支撑树 / 生成树

给定一个连通赋权无向图，若一个子图满足：

1. 包含原图的全部顶点；
2. 连通；
3. 没有圈；
4. 边数为 `n-1`，其中 `n` 为顶点数；

则称它为原图的一棵支撑树，也叫生成树。

## 2. 最小权支撑树

在所有支撑树中，边权之和最小的那一棵称为最小权支撑树。

```text
目标：连接全部顶点，但总权数最小，并且不能形成圈。
```

## 3. 本图中的顶点和边

按位置给图中7个点编号：

```text
A: 左上
B: 左下
C: 中左
D: 上中
E: 下中
F: 中右
G: 最右
```

边和权数：

```text
AB=3
AC=2
BC=7
AD=7
BE=4
CF=1
DF=2
EF=2
DG=4
FG=4
EG=5
```

共有7个顶点，因此最小权支撑树需要选：

```text
7-1=6 条边
```

## 4. 按避圈法 / Kruskal 思想求解

按权数从小到大排序，依次选边，但不能形成圈。

排序：

```text
CF=1
AC=2
DF=2
EF=2
AB=3
BE=4
DG=4
FG=4
EG=5
AD=7
BC=7
```

依次选择：

1. 选 `CF=1`。
2. 选 `AC=2`，不成圈。
3. 选 `DF=2`，不成圈。
4. 选 `EF=2`，不成圈。
5. 选 `AB=3`，把B接进来，不成圈。
6. 选 `DG=4` 或 `FG=4`，把G接进来。

已经选满6条边，停止。

一种最小权支撑树为：

```text
CF, AC, DF, EF, AB, DG
```

总权数：

```text
1+2+2+2+3+4=14
```

也可以把最后一条 `DG=4` 换成 `FG=4`，总权数仍为14。

## 5. 结论

本图的最小权支撑树权数为：

```text
14
```

记忆：

```text
最小权支撑树 = 连通所有点 + 不成圈 + 总权最小
n个顶点的支撑树一定有 n-1 条边
```
