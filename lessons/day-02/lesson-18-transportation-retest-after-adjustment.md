# 第四章 运输问题：修正后还要继续做最优性检验

修正后方案：

```text
        1   2   3   4 | ai
1       -   -   5   2 | 7
2       3   -   -   1 | 4
3       -   6   -   3 | 9
-------------------------
bj      3   6   5   6
```

即基变量格为：

```text
(1,3), (1,4), (2,1), (2,4), (3,2), (3,4)
```

运输问题每修正一次，都需要重新用位势法检验一次。只有所有空格检验数都满足课件最优条件，才能停止。

## 1. 重新求位势

对基变量格列方程：

```text
u1+v3 = c13 = 3
u1+v4 = c14 = 10
u2+v1 = c21 = 1
u2+v4 = c24 = 8
u3+v2 = c32 = 4
u3+v4 = c34 = 5
```

令：

```text
u1=0
```

解得：

```text
v3=3
v4=10
u2=-2
v1=3
u3=-5
v2=9
```

所以：

```text
u = (0,-2,-5)
v = (3,9,3,10)
```

## 2. 按课件公式求空格检验数

课件公式：

```text
r_ij = u_i + v_j - c_ij
```

空格为：

```text
(1,1), (1,2), (2,2), (2,3), (3,1), (3,3)
```

逐个计算：

```text
r11 = 0+3-3 = 0
r12 = 0+9-11 = -2
r22 = -2+9-9 = -2
r23 = -2+3-2 = -1
r31 = -5+3-7 = -9
r33 = -5+3-10 = -12
```

## 3. 判断

课件写法下，最小化运输问题的最优条件为：

```text
所有空格 r_ij <= 0
```

本例所有空格检验数都满足：

```text
0, -2, -2, -1, -9, -12 <= 0
```

所以修正后的方案已经是最优解。

其中 `r11=0` 表示可能存在另一个同样最优的运输方案，但当前方案已经达到最小总费用。

最优运输费用：

```text
z = 85
```
