# 第四章 运输问题：为什么空格不满足 ui+vj=cij

课件求位势时使用：

```text
u_i + v_j = c_ij    (t_ij ∈ Δ)
```

注意：这个等式只对基变量格成立，也就是当前有红圈运输量的格子。

当前基变量格：

```text
(1,3), (1,4), (2,1), (2,3), (3,2), (3,4)
```

所以只列这些方程：

```text
u1+v3=3
u1+v4=10
u2+v1=1
u2+v3=2
u3+v2=4
u3+v4=5
```

令 `u1=0` 后，得：

```text
v3=3
v4=10
u2=-1
v1=2
u3=-5
v2=9
```

空格 `(2,4)` 没有红圈运输量，不属于基变量格，所以不能要求：

```text
u2+v4=c24
```

它应当用来计算检验数：

```text
r24 = u2+v4-c24
    = -1+10-8
    = 1
```

因此，`-1+10` 不需要等于 `8`；它们的差值就是空格 `(2,4)` 的检验数。

若检验数为正，表示该空格可以进入基变量，当前方案还不是最优。
