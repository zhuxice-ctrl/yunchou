# 第四章 运输问题：修正后最优性检验表格版

修正后运输方案：

```text
        1   2   3   4 | ai
1       -   -   5   2 | 7
2       3   -   -   1 | 4
3       -   6   -   3 | 9
-------------------------
bj      3   6   5   6
```

基变量格：

```text
(1,3), (1,4), (2,1), (2,4), (3,2), (3,4)
```

## 1. 求位势表

| 基变量格 | 方程 | 已知 | 求出 |
|---|---|---|---|
| (1,3) | u1+v3=3 | u1=0 | v3=3 |
| (1,4) | u1+v4=10 | u1=0 | v4=10 |
| (2,4) | u2+v4=8 | v4=10 | u2=-2 |
| (2,1) | u2+v1=1 | u2=-2 | v1=3 |
| (3,4) | u3+v4=5 | v4=10 | u3=-5 |
| (3,2) | u3+v2=4 | u3=-5 | v2=9 |

位势结果：

```text
u1=0, u2=-2, u3=-5
v1=3, v2=9, v3=3, v4=10
```

## 2. 空格检验数表

课件公式：

```text
r_ij = u_i + v_j - c_ij
```

| 空格 | 计算 | 检验数 |
|---|---|---|
| (1,1) | 0+3-3 | 0 |
| (1,2) | 0+9-11 | -2 |
| (2,2) | -2+9-9 | -2 |
| (2,3) | -2+3-2 | -1 |
| (3,1) | -5+3-7 | -9 |
| (3,3) | -5+3-10 | -12 |

## 3. 判断表

| 检验数 | 是否满足 r_ij <= 0 |
|---|---|
| r11=0 | 满足 |
| r12=-2 | 满足 |
| r22=-2 | 满足 |
| r23=-1 | 满足 |
| r31=-9 | 满足 |
| r33=-12 | 满足 |

结论：所有空格检验数均满足 `r_ij<=0`，所以修正后方案为最优方案。

```text

```
