# 第四章 4.3 运输问题：位势与检验数

课件使用的检验数公式为：

```text
r_ij = u_i + v_j - c_ij    (t_ij 不属于 Δ)
```

其中：

- `c_ij` 是单位运价，即每个格子右上角的原始运价；
- 红圈数字是当前运输量 `x_ij`；
- `u_i` 是行位势；
- `v_j` 是列位势；
- `r_ij` 是空格的检验数。

当前基变量格为：

```text
(1,3), (1,4), (2,1), (2,3), (3,2), (3,4)
```

对基变量格列方程：

```text
u_i + v_j = c_ij
```

即：

```text
u1+v3=3
u1+v4=10
u2+v1=1
u2+v3=2
u3+v2=4
u3+v4=5
```

令 `u1=0`，解得：

```text
v3=3
v4=10
u2=-1
v1=2
u3=-5
v2=9
```

所以课件表头为：

```text
v_j: 2, 9, 3, 10
u_i: 0, -1, -5
```

空格检验数：

```text
r11 = 0+2-3 = -1
r12 = 0+9-11 = -2
r22 = -1+9-9 = -1
r24 = -1+10-8 = 1
r31 = -5+2-7 = -10
r33 = -5+3-10 = -12
```

因为课件用的是 `r_ij = u_i+v_j-c_ij`，最小化运输问题的最优条件是：

```text
所有空格 r_ij <= 0
```

若存在正检验数，则当前方案不是最优。

本例中：

```text
r24=1>0
```

因此当前方案不是最优，下一步选 `(2,4)` 入基。

注意：若使用另一种写法 `sigma_ij=c_ij-(u_i+v_j)`，则符号正好相反：

```text
sigma_ij = - r_ij
```

本课程以后按课件写法使用 `r_ij = u_i+v_j-c_ij`。
