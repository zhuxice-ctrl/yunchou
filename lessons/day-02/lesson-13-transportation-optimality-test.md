# 第四章 4.3 运输问题：最优性检验

最小元素法得到的初始方案：

```text
        1   2   3   4 | ai
1       -   -   4   3 | 7
2       3   -   1   - | 4
3       -   6   -   3 | 9
-------------------------
bj      3   6   5   6
```

即：

```text
x13=4, x14=3
x21=3, x23=1
x32=6, x34=3
```

单位运价：

```text
c11=3,  c12=11, c13=3,  c14=10
c21=1,  c22=9,  c23=2,  c24=8
c31=7,  c32=4,  c33=10, c34=5
```

## 位势法检验

对所有基变量格 `(i,j)` 建立：

```text
ui + vj = cij
```

基变量格为：

```text
(1,3), (1,4), (2,1), (2,3), (3,2), (3,4)
```

所以：

```text
u1+v3=3
u1+v4=10
u2+v1=1
u2+v3=2
u3+v2=4
u3+v4=5
```

令 `u1=0`，解得：

```text
v3=3
v4=10
u2=-1
v1=2
u3=-5
v2=9
```

对所有空格计算检验数：

```text
sigma_ij = cij - (ui + vj)
```

空格检验数：

```text
sigma11 = 3-(0+2)=1
sigma12 = 11-(0+9)=2
sigma22 = 9-(-1+9)=1
sigma24 = 8-(-1+10)=-1
sigma31 = 7-(-5+2)=10
sigma33 = 10-(-5+3)=12
```

因为最小化运输问题要求所有空格检验数都 `>=0` 才最优，而：

```text
sigma24 = -1 < 0
```

所以当前方案不是最优解。

## 选入基格并调整

选 `sigma24` 对应的空格 `(2,4)` 入基。

闭回路为：

```text
(2,4) -> (2,3) -> (1,3) -> (1,4) -> (2,4)
```

按交替正负号：

```text
+ (2,4)
- (2,3)
+ (1,3)
- (1,4)
```

负号格的运输量为：

```text
x23=1
x14=3
```

所以：

```text
theta = min(1,3)=1
```

调整：

```text
x24 = 0+1 = 1
x23 = 1-1 = 0
x13 = 4+1 = 5
x14 = 3-1 = 2
```

其他保持不变：

```text
x21=3, x32=6, x34=3
```

新方案：

```text
        1   2   3   4 | ai
1       -   -   5   2 | 7
2       3   -   -   1 | 4
3       -   6   -   3 | 9
-------------------------
bj      3   6   5   6
```

新总运费：

```text
z = 5*3 + 2*10 + 3*1 + 1*8 + 6*4 + 3*5
  = 85
```

继续检验，若所有检验数均非负，则新方案最优。
