# 第五章 指派问题：排课表问题

原始费用矩阵：

```text
        微积分  线性代数  概率论  运筹学
A         2      10       9      7
B        15       4      14      8
C        13      14      16     11
D         4      15      13      9
```

目标：每位教师上一门课，每门课只安排一位教师，使总课时费最小。

## 1. 行约简

各行最小值：

```text
A: 2
B: 4
C: 11
D: 4
```

每行减去本行最小值，得到：

```text
0   8   7   5
11  0  10   4
2   3   5   0
0  11   9   5
```

## 2. 列约简

各列最小值：

```text
第1列: 0
第2列: 0
第3列: 5
第4列: 0
```

第3列减5，其他列不变，得到约化费用矩阵：

```text
0   8   2   5
11  0   5   4
2   3   0   0
0  11   4   5
```

## 3. 第一次寻找独立零元素

零元素位置：

```text
A: 微积分
B: 线性代数
C: 概率论、运筹学
D: 微积分
```

由于 A 与 D 都只能选微积分列，无法得到4个独立零元素，所以转入画线调整。

## 4. 画最少直线并调整矩阵

用3条线覆盖所有零元素，例如：

```text
第1列、B行、C行
```

未被直线覆盖的元素为：

```text
A行第2、3、4列: 8,2,5
D行第2、3、4列: 11,4,5
```

其中最小值：

```text
theta = 2
```

按课件步骤：

- 未被直线覆盖的元素全部减去2；
- 两条直线交叉处的元素加上2；
- 其余元素不变。

得到新矩阵：

```text
0   6   0   3
13  0   5   4
4   3   0   0
0   9   2   3
```

## 5. 再次寻找独立零元素

零元素位置：

```text
A: 微积分、概率论
B: 线性代数
C: 概率论、运筹学
D: 微积分
```

可以圈出4个独立零元素：

```text
A -> 概率论
B -> 线性代数
C -> 运筹学
D -> 微积分
```

因为独立零元素个数等于矩阵阶数4，所以得到最优指派方案。

## 6. 回到原始费用矩阵计算总费用

```text
A -> 概率论: 9
B -> 线性代数: 4
C -> 运筹学: 11
D -> 微积分: 4
```

总费用：

```text
9 + 4 + 11 + 4 = 28
```

最优排课方案：

```text
A 教概率论
B 教线性代数
C 教运筹学
D 教微积分
最小总课时费 = 28
```
