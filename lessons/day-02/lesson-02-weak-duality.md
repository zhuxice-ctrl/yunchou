# 第二章 第2课：性质2——弱对偶性

> 本讲严格按照教师课件《2.2 对偶问题的基本性质》中“性质2”解释。

## 一、课件中的原问题与对偶问题

原问题可写为：

```text
max c^T x
s.t. Ax <= b, x >= 0
```

对偶问题可写为：

```text
min b^T y
s.t. A^T y >= c, y >= 0
```

其中：

- `K(P)`：原问题可行解集合；
- `K(D)`：对偶问题可行解集合；
- `x in K(P)`：x满足原问题全部约束；
- `y in K(D)`：y满足对偶问题全部约束。

## 二、性质2（弱对偶性）

对任意：

```text
x in K(P), y in K(D)
```

都有：

```text
c^T x <= b^T y
```

含义：

- 原问题是最大化问题；
- 对偶问题是最小化问题；
- 任意原问题可行解的目标值，都不会超过任意对偶问题可行解的目标值。

所以：

```text
原问题可行值 <= 对偶问题可行值
```

## 三、为什么成立

因为：

```text
Ax <= b, y >= 0
```

所以：

```text
y^T Ax <= y^T b = b^T y
```

又因为：

```text
A^T y >= c, x >= 0
```

所以：

```text
x^T A^T y >= x^T c = c^T x
```

而：

```text
x^T A^T y = y^T Ax
```

因此：

```text
c^T x <= x^T A^T y = y^T Ax <= b^T y
```

最终得到：

```text
c^T x <= b^T y
```

## 四、目标值相等的条件

课件给出：

```text
c^T x = b^T y
```

当且仅当：

```text
(b-Ax)^T y = 0
x^T(A^T y-c) = 0
```

其中：

- `b-Ax`：原问题各约束的松弛量；
- `A^T y-c`：对偶问题各约束的松弛量。

两个式子都等于0，表示：

```text
松弛量与对应变量不能同时为正
```

也就是：

```text
原约束有剩余  => 对应对偶变量为0
对偶变量大于0 => 对应原约束必须取等号

对偶约束有剩余 => 对应原变量为0
原变量大于0   => 对应对偶约束必须取等号
```

## 五、等价分解

课件中的等价条件来自：

```text
b^T y-c^T x
= y^T(b-Ax)+x^T(A^T y-c)
```

右边两项都大于等于0。

因此若：

```text
b^T y-c^T x = 0
```

就必须同时有：

```text
y^T(b-Ax)=0
x^T(A^T y-c)=0
```

反过来，两项都为0，则目标值必相等。

## 六、课件“推论”为什么只剩一项

在课件推论所采用的标准形式中，原问题满足：

```text
Ax = b
```

因此：

```text
b-Ax = 0
```

第一项自动为0，于是只需判断：

```text
x^T(A^T y-c)=0
```

## 七、最重要的考试结论

```text
任意原可行解目标值 <= 任意对偶可行解目标值
```

若能找到一对可行解使：

```text
c^T x = b^T y
```

那么这两个解分别就是原问题和对偶问题的最优解。
