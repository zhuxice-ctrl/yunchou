# 第1课：1.1 线性规划问题与建模

## 学习目标

完成本课后，你需要能够：

- 识别决策变量、目标函数和约束条件；
- 把简单文字题翻译成线性规划模型；
- 解释可行解、可行域、最优解和最优值。

## 1. 什么是线性规划

线性规划研究的是：

> 在一组线性约束条件下，寻找一组决策变量的取值，使一个线性目标函数达到最大或最小。

可以把它理解为：

- 你可以做哪些选择；
- 这些选择受到哪些限制；
- 在限制内怎样做到最好。

## 2. 一个线性规划模型的四个部分

### 2.1 决策变量

决策变量表示“我们需要决定什么”。

例如某工厂生产甲、乙两种产品，可以设：

- `x1`：甲产品的产量；
- `x2`：乙产品的产量。

### 2.2 目标函数

目标函数表示“我们希望最大化或最小化什么”。

例如甲每件利润40元，乙每件利润30元，则总利润为：

```text
z = 40x1 + 30x2
```

若目标是利润最大，则写成：

```text
max z = 40x1 + 30x2
```

### 2.3 约束条件

约束条件表示资源、时间、资金、市场等限制。

例如：

- 甲每件耗用2单位原料，乙每件耗用1单位原料，总原料不超过100；
- 甲每件耗用1小时，乙每件耗用2小时，总工时不超过80。

则约束为：

```text
2x1 + x2 <= 100
x1 + 2x2 <= 80
```

### 2.4 非负约束

产量通常不能为负，所以还需要：

```text
x1 >= 0
x2 >= 0
```

## 3. 完整例题

某工厂生产甲、乙两种产品。

| 项目 | 甲 | 乙 | 可用量 |
|---|---:|---:|---:|
| 原料 | 2 | 1 | 100 |
| 工时 | 1 | 2 | 80 |
| 单位利润 | 40 | 30 | — |

问：怎样安排生产，使总利润最大？

### 第一步：定义变量

设：

```text
x1 = 甲产品产量
x2 = 乙产品产量
```

### 第二步：写目标函数

```text
max z = 40x1 + 30x2
```

### 第三步：写约束条件

原料约束：

```text
2x1 + x2 <= 100
```

工时约束：

```text
x1 + 2x2 <= 80
```

非负约束：

```text
x1 >= 0, x2 >= 0
```

### 最终模型

```text
max z = 40x1 + 30x2
s.t.
    2x1 + x2 <= 100
    x1 + 2x2 <= 80
    x1, x2 >= 0
```

## 4. 四个基本概念

### 可行解

同时满足所有约束条件的一组变量取值。

例如 `(x1, x2) = (20, 20)`：

```text
2×20 + 20 = 60 <= 100
20 + 2×20 = 60 <= 80
```

所以它是可行解。

### 不可行解

只要违反任意一个约束，就不是可行解。

例如 `(50, 20)`：

```text
2×50 + 20 = 120 > 100
```

违反原料约束，因此不可行。

### 可行域

所有可行解组成的集合。

### 最优解与最优值

- 最优解：让目标函数达到最好结果的变量取值；
- 最优值：该最优解对应的目标函数数值。

## 5. 判断一个模型是不是线性规划

必须同时满足：

1. 目标函数是线性的；
2. 约束条件是线性的；
3. 变量之间没有乘积、平方、开方、绝对值等非线性形式。

线性例子：

```text
3x1 + 2x2
```

非线性例子：

```text
x1 × x2
x1^2
sqrt(x1)
```

## 6. 建模固定模板

拿到文字题后，按下面顺序写：

```text
1. 设决策变量
2. 写目标函数
3. 写资源或业务约束
4. 写非负约束或整数约束
5. 检查单位和方向
```

## 7. 常见错误

- 没有定义变量；
- 把“最多”写成 `>=`；
- 把“至少”写成 `<=`；
- 忘记非负约束；
- 目标函数系数与利润、成本对应错误；
- 约束中不同单位直接相加。

## 8. 本课小结

一个标准线性规划模型可以概括为：

```text
在约束条件下，让目标函数最大或最小。
```

建模时始终抓住三个问题：

- 决定什么？
- 限制是什么？
- 优化什么？
