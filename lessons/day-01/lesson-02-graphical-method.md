# 第2课：1.2 图解法

## 学习目标

完成本课后，应能：

1. 把二元线性规划约束画在坐标平面上。
2. 找出可行域。
3. 求可行域顶点。
4. 用目标函数判断最优解。
5. 识别唯一最优、无穷多最优、无界和不可行。

## 一、图解法适用范围

图解法主要用于只有两个决策变量的线性规划问题。

例如：

```text
max z = 3x1 + 2x2

s.t.
    x1 + x2 <= 4
    x1 <= 3
    x2 <= 2
    x1, x2 >= 0
```

## 二、基本步骤

```mermaid
flowchart LR
    A[把约束变成边界直线] --> B[判断每个约束的可行半平面]
    B --> C[取所有半平面的交集]
    C --> D[求可行域顶点]
    D --> E[代入目标函数比较]
    E --> F[确定最优解和最优值]
```

## 三、怎样画一条约束直线

以

```text
x1 + x2 <= 4
```

为例，先画边界直线：

```text
x1 + x2 = 4
```

令 `x1 = 0`，得 `x2 = 4`；令 `x2 = 0`，得 `x1 = 4`。

因此直线经过 `(0,4)` 和 `(4,0)`。

然后判断哪一侧满足不等式。最方便的方法是代入原点 `(0,0)`：

```text
0 + 0 <= 4
```

成立，所以包含原点的一侧就是可行半平面。

## 四、完整例题

求解：

```text
max z = 3x1 + 2x2

s.t.
    x1 + x2 <= 4
    x1 <= 3
    x2 <= 2
    x1, x2 >= 0
```

### 1. 画边界线

- `x1 + x2 = 4`
- `x1 = 3`
- `x2 = 2`
- 坐标轴 `x1 = 0`、`x2 = 0`

### 2. 找可行域

所有约束同时满足的区域，其顶点为：

```text
(0,0), (3,0), (3,1), (2,2), (0,2)
```

### 3. 代入目标函数

| 顶点 | z = 3x1 + 2x2 |
|---|---:|
| (0,0) | 0 |
| (3,0) | 9 |
| (3,1) | 11 |
| (2,2) | 10 |
| (0,2) | 4 |

所以：

```text
最优解：(x1, x2) = (3,1)
最优值：z = 11
```

## 五、为什么只检查顶点

若线性规划存在最优解，则至少可以在可行域顶点中找到一个最优解。

因此考试中可直接：

1. 求可行域顶点；
2. 代入目标函数；
3. 比较大小。

## 六、四种特殊情况

### 1. 唯一最优解

只有一个顶点取得最佳目标值。

### 2. 无穷多最优解

目标函数等值线与可行域某条边平行，并且该边上的两个端点都取得相同最优值，则整条线段上的点都是最优解。

### 3. 无界

可行域沿目标函数增大的方向无限延伸，目标值可以无限增大或减小，因此没有有限最优值。

### 4. 不可行

所有约束对应的半平面没有公共交集，因此不存在可行解。

## 七、考试作答模板

```text
1. 作各约束对应的边界直线；
2. 根据不等号方向确定可行半平面；
3. 得到可行域并求其顶点；
4. 将各顶点代入目标函数；
5. 比较目标函数值，得到最优解和最优值。
```

## 八、常见错误

1. 直接画不等式，不先画等式边界线。
2. 半平面方向判断错误。
3. 漏掉非负约束。
4. 漏算两条直线的交点。
5. 只凭图形猜最优点，不代入目标函数验证。
6. 把最优解和最优值混淆。
