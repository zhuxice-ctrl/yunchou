# 第3课：线性规划标准形 + 基本解（单纯形法前置核心）

## 学习目标

完成本课后，应能：

1. 理解什么是“标准形”。
2. 会把一般线性规划转成标准形。
3. 理解“基变量、非基变量”。
4. 理解“基本解”和“基本可行解”。
5. 为后续单纯形法做准备。

---

# 一、为什么要学标准形

单纯形法只能处理一种固定格式的模型：

👉 这就是“标准形”。

所以任何题目最终都必须变成标准形。

---

# 二、标准形定义

一个线性规划的标准形一般写作：

```text
max z = c1x1 + c2x2 + ... + cnxn

s.t.
    a11x1 + a12x2 + ... + a1nxn <= b1
    a21x1 + a22x2 + ... + a2nxn <= b2
    ...
    xi >= 0
```

关键特征：

- 目标函数：最大化
- 约束：全部 ≤ 型
- 变量：全部 ≥ 0

---

# 三、如何把模型变成标准形

## 1. 目标函数统一为 max

如果是 min：

```text
min z → max (-z)
```

---

## 2. 不等式统一为 ≤

三种情况：

### (1) ≤ 不变

```text
x1 + x2 <= 5
```

直接保留。

---

### (2) ≥ 变号

```text
x1 + x2 >= 5
```

两边乘 -1：

```text
-x1 - x2 <= -5
```

---

### (3) = 拆成两个

```text
x1 + x2 = 5
```

等价于：

```text
x1 + x2 <= 5
x1 + x2 >= 5
```

再继续变换。

---

## 3. 变量必须非负

如果出现自由变量：

```text
x = x+ - x-
```

其中：

```text
x+, x- >= 0
```

---

# 四、标准形例题

原问题：

```text
min z = x1 - 2x2

s.t.
    x1 + x2 >= 4
    x1 - x2 = 2
```

---

## Step 1：目标函数变 max

```text
max z = -x1 + 2x2
```

---

## Step 2：约束统一 ≤

### 第一条：

```text
x1 + x2 >= 4
→ -x1 - x2 <= -4
```

---

### 第二条：等式

```text
x1 - x2 = 2
```

拆成：

```text
x1 - x2 <= 2
x1 - x2 >= 2 → -x1 + x2 <= -2
```

---

## 五、基本解（非常重要）

### 1. 什么是基本解？

在标准形中：

👉 选出一部分变量设为 0，解剩下变量

---

### 2. 基变量 vs 非基变量

- 非基变量：设为 0
- 基变量：由方程解出来

---

## 六、举例理解基本解

假设：

```text
x1 + x2 + x3 = 4
2x1 + x2 + x4 = 6
```

有 4 个变量，2 个方程。

👉 选 2 个变量为基变量

例如：

令 x3 = 0, x4 = 0

解：

```text
x1 + x2 = 4
2x1 + x2 = 6
```

解得：

```text
x1 = 2
x2 = 2
```

所以：

```text
(x1, x2, x3, x4) = (2,2,0,0)
```

这就是一个**基本解**。

---

## 七、基本可行解

如果基本解满足：

```text
所有 xi >= 0
```

那么它就是：

👉 基本可行解（BFS）

---

## 八、为什么 BFS 很重要

单纯形法的核心思想是：

> 在所有基本可行解之间跳跃，找到最优解

---

## 九、本课核心总结

你必须记住三句话：

```text
标准形：单纯形法的输入格式
基本解：选部分变量为0后解方程
基本可行解：满足非负的基本解
```

## 十、常见错误

1. 忘记把 min 转 max
2. 忘记变量非负
3. 把 = 直接当 ≤
4. 不理解“设为0”的意义
5. 基变量选错数量
