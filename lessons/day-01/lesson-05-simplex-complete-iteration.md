# 第5课：按照课件格式学习单纯形法

## 课件中的例题表

课件采用简式单纯形表：左侧写基变量 `x_B`，最后一行写 `Z`，右端常数写 `b̄`。

初始表：

| x_B | x1 | x2 | x3 | x4 | x5 | b̄ |
|---|---:|---:|---:|---:|---:|---:|
| x3 | 2 | 2 | 1 | 0 | 0 | 12 |
| x4 | 1 | 2 | 0 | 1 | 0 | 8 |
| x5 | 4 | 0 | 0 | 0 | 1 | 16 |
| Z | 2 | 3 | 0 | 0 | 0 | 0 |

这张表对应：

```text
max Z = 2x1 + 3x2

s.t.
    2x1 + 2x2 + x3 = 12
    x1 + 2x2 + x4 = 8
    4x1 + x5 = 16
    xi >= 0
```

初始基变量是 `x3、x4、x5`，所以初始基本可行解为：

```text
x1 = 0, x2 = 0
x3 = 12, x4 = 8, x5 = 16
Z = 0
```

## 第一步：选入基变量

最大化问题中，看 `Z` 行的正数：

```text
x1 列为 2
x2 列为 3
```

最大正数是3，所以：

```text
x2 入基
```

## 第二步：选出基变量

对 `x2` 列做最小正比值：

```text
x3 行：12 / 2 = 6
x4 行： 8 / 2 = 4
x5 行：系数为0，不参与比值
```

最小正比值是4，所以：

```text
x4 出基
```

主元是 `x4` 行、`x2` 列交点处的2。

## 第三步：转轴

主元行除以2：

```text
新R2 = 旧R2 / 2
```

得到：

```text
(1/2, 1, 0, 1/2, 0 | 4)
```

然后把 `x2` 列其他元素消成0：

```text
新R1 = 旧R1 - 2新R2
新R3 = 旧R3
新Z行 = 旧Z行 - 3新R2
```

转轴后：

| x_B | x1 | x2 | x3 | x4 | x5 | b̄ |
|---|---:|---:|---:|---:|---:|---:|
| x3 | 1 | 0 | 1 | -1 | 0 | 4 |
| x2 | 1/2 | 1 | 0 | 1/2 | 0 | 4 |
| x5 | 4 | 0 | 0 | 0 | 1 | 16 |
| Z | 1/2 | 0 | 0 | -3/2 | 0 | -12 |

当前基变量为：

```text
x3 = 4, x2 = 4, x5 = 16
```

非基变量为：

```text
x1 = 0, x4 = 0
```

当前目标值：

```text
Z = 12
```

注意：课件表右下角写的是 `-12`，读取实际目标值时取相反数。

## 是否已经最优

看新的 `Z` 行：

```text
x1 列仍有正数 1/2
```

所以当前还没有最优，下一轮应让：

```text
x1 入基
```

## 课件版固定规则

```text
1. 看Z行，最大正数对应变量入基；
2. 对入基列做最小正比值；
3. 最小正比值所在行的基变量出基；
4. 交叉位置是主元；
5. 主元化1，同列其他元素化0；
6. Z行全部<=0时停止；
7. 右下角若记录-Z，实际目标值取相反数。
```
