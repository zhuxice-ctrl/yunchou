# 第5课：单纯形法完整迭代

## 例题

```text
max z = 3x1 + 2x2

s.t.
    x1 + x2 <= 4
    2x1 + x2 <= 6
    x1, x2 >= 0
```

加入松弛变量：

```text
x1 + x2 + x3 = 4
2x1 + x2 + x4 = 6
```

初始基本可行解：

```text
x1 = 0, x2 = 0, x3 = 4, x4 = 6
```

## 初始表

| Cb | 基变量 | b | x1 | x2 | x3 | x4 |
|---:|---|---:|---:|---:|---:|---:|
| 0 | x3 | 4 | 1 | 1 | 1 | 0 |
| 0 | x4 | 6 | 2 | 1 | 0 | 1 |
|  | Cj-Zj |  | 3 | 2 | 0 | 0 |

### 第一次迭代

- 最大正检验数为3，所以 `x1` 入基。
- 比值：`4/1=4`，`6/2=3`。
- 最小正比值为3，所以 `x4` 出基。
- 主元为2。

主元行除以2：

```text
R2 = R2 / 2
```

第一行消元：

```text
R1 = R1 - R2
```

得到：

| Cb | 基变量 | b | x1 | x2 | x3 | x4 |
|---:|---|---:|---:|---:|---:|---:|
| 0 | x3 | 1 | 0 | 0.5 | 1 | -0.5 |
| 3 | x1 | 3 | 1 | 0.5 | 0 | 0.5 |
|  | Cj-Zj |  | 0 | 0.5 | 0 | -1.5 |

### 第二次迭代

- `x2` 的检验数为0.5，大于0，所以 `x2` 入基。
- 比值：`1/0.5=2`，`3/0.5=6`。
- 最小正比值为2，所以 `x3` 出基。
- 主元为0.5。

主元行除以0.5：

```text
R1 = R1 / 0.5
```

得到：

```text
R1: b=2, x1=0, x2=1, x3=2, x4=-1
```

第二行消元：

```text
R2 = R2 - 0.5R1
```

得到最终表：

| Cb | 基变量 | b | x1 | x2 | x3 | x4 |
|---:|---|---:|---:|---:|---:|---:|
| 2 | x2 | 2 | 0 | 1 | 2 | -1 |
| 3 | x1 | 2 | 1 | 0 | -1 | 1 |
|  | Cj-Zj |  | 0 | 0 | -1 | -1 |

因为所有检验数都小于等于0，所以达到最优。

## 最优解

```text
x1 = 2
x2 = 2
```

松弛变量：

```text
x3 = 0
x4 = 0
```

最优值：

```text
z = 3×2 + 2×2 = 10
```

## 单纯形法考试模板

```text
1. 加松弛变量，化成等式；
2. 建立初始单纯形表；
3. 用检验数选入基变量；
4. 用最小正比值选出基变量；
5. 主元行归一化；
6. 入基列其余元素消为0；
7. 重新计算检验数；
8. 若仍有正检验数，继续迭代；
9. 所有检验数<=0时读出最优解。
```
