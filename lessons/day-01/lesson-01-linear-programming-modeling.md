# 第1课：1.1 线性规划问题与建模

## 学习目标

完成本课后，应能：

1. 理解决策变量、目标函数、约束条件。
2. 判断一个问题是否属于线性规划。
3. 把简单文字题翻译成线性规划模型。
4. 理解可行解、可行域、最优解和最优值。

## 一、什么是运筹学

运筹学关注的是：在资源有限、条件受限的情况下，怎样作出更好的决策。

例如：

- 工厂如何安排生产才能利润最大；
- 运输公司如何安排路线才能费用最低；
- 仓库如何分配库存才能减少成本。

## 二、线性规划的四个核心组成

### 1. 决策变量

决策变量表示“我们需要决定什么”。

例如：

- `x1`：生产产品甲的数量；
- `x2`：生产产品乙的数量。

### 2. 目标函数

目标函数表示“我们希望什么达到最好”。

常见目标：

- 利润最大；
- 成本最小；
- 时间最短；
- 距离最短。

例如：

`max z = 40x1 + 30x2`

表示每生产一个甲产品获利40元，每生产一个乙产品获利30元，希望总利润最大。

### 3. 约束条件

约束条件表示“决策受到哪些限制”。

例如：

`2x1 + x2 <= 100`

表示两种产品共同消耗某种资源，而且资源总量最多为100。

### 4. 非负约束

实际数量通常不能为负，所以写成：

`x1 >= 0, x2 >= 0`

## 三、线性规划的一般结构

```text
max 或 min  z = c1x1 + c2x2 + ... + cnxn

s.t.
    a11x1 + a12x2 + ... + a1nxn <= / = / >= b1
    a21x1 + a22x2 + ... + a2nxn <= / = / >= b2
    ...
    x1, x2, ..., xn >= 0
```

其中：

- `x` 是决策变量；
- `c` 是目标函数系数；
- `a` 是资源消耗系数；
- `b` 是资源总量或限制值。

## 四、怎样判断是不是线性规划

线性规划要求目标函数和约束条件都是线性的。

### 线性形式

- `3x1 + 2x2`
- `x1 - 4x2 <= 10`

### 非线性形式

- `x1^2 + x2`
- `x1x2`
- `sqrt(x1)`
- `1/x1`

判断口诀：

> 变量只能是一次方，变量之间不能相乘，变量不能出现在分母、根号或指数中。

## 五、完整例题

某工厂生产甲、乙两种产品。每件甲产品利润40元，每件乙产品利润30元。生产一件甲需要2小时机器时间和1千克原料；生产一件乙需要1小时机器时间和2千克原料。每天最多有100小时机器时间和80千克原料。问怎样安排生产，才能使利润最大？

### 第一步：确定决策变量

设：

- `x1` 为甲产品产量；
- `x2` 为乙产品产量。

### 第二步：写目标函数

总利润为：

`40x1 + 30x2`

所以：

`max z = 40x1 + 30x2`

### 第三步：写约束条件

机器时间约束：

`2x1 + x2 <= 100`

原料约束：

`x1 + 2x2 <= 80`

非负约束：

`x1 >= 0, x2 >= 0`

### 最终模型

```text
max z = 40x1 + 30x2

s.t.
    2x1 + x2 <= 100
    x1 + 2x2 <= 80
    x1, x2 >= 0
```

## 六、四个重要概念

### 可行解

满足所有约束条件的一组变量取值。

例如 `(x1, x2) = (20, 20)`：

- `2×20 + 20 = 60 <= 100`
- `20 + 2×20 = 60 <= 80`

所以它是可行解。

### 不可行解

只要违反任意一个约束，就不是可行解。

例如 `(50, 20)`：

- `2×50 + 20 = 120 > 100`

因此不可行。

### 可行域

所有可行解组成的集合。

### 最优解与最优值

- 最优解：使目标函数达到最佳的可行解；
- 最优值：该最优解对应的目标函数值。

## 七、建模五步法

```mermaid
flowchart LR
    A[读题] --> B[找要决定的量]
    B --> C[设决策变量]
    C --> D[写目标函数]
    D --> E[逐条写约束]
    E --> F[补非负约束]
```

## 八、常见错误

1. 决策变量没有说明单位。
2. 目标函数方向写错，利润应最大、成本通常应最小。
3. 约束不等号方向写反。
4. 漏写非负约束。
5. 把资源总量写进目标函数。
6. 把单件消耗量与总消耗量混淆。

## 九、本课小结

线性规划模型最核心的结构是：

```text
决定什么 + 优化什么 + 受什么限制
```

对应数学语言：

```text
决策变量 + 目标函数 + 约束条件
```
