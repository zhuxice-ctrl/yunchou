# 第1课练习：线性规划问题与建模

## 使用方法

先独立完成，再查看答案文件。建议限时25分钟。

## 练习1：识别模型要素

某饮料店制作A、B两种饮料。A每杯利润8元，B每杯利润6元。每天最多使用牛奶1200毫升和糖浆600毫升。A每杯需要牛奶200毫升、糖浆50毫升；B每杯需要牛奶100毫升、糖浆100毫升。

请写出：

1. 决策变量；
2. 目标函数；
3. 约束条件；
4. 非负约束。

## 练习2：判断可行解

考虑模型：

```text
max z = 5x1 + 4x2
s.t.
    2x1 + x2 <= 10
    x1 + 2x2 <= 8
    x1, x2 >= 0
```

判断以下点是否为可行解，并说明理由：

1. `(2, 2)`
2. `(4, 2)`
3. `(-1, 3)`

## 练习3：判断是否为线性规划

判断下列模型是不是线性规划，并说明原因。

### A

```text
max z = 3x1 + 2x2
s.t.
    x1 + x2 <= 10
    x1, x2 >= 0
```

### B

```text
max z = x1 × x2
s.t.
    x1 + x2 <= 10
    x1, x2 >= 0
```

### C

```text
min z = 2x1 + 5x2
s.t.
    x1^2 + x2 <= 20
    x1, x2 >= 0
```

## 练习4：独立建模

某公司运输甲、乙两种货物。

| 项目 | 甲 | 乙 | 上限 |
|---|---:|---:|---:|
| 体积 | 3 | 2 | 18 |
| 重量 | 2 | 4 | 20 |
| 单位利润 | 50 | 40 | — |

建立使总利润最大的线性规划模型。
