# 第1课练习：线性规划建模

## 练习1：识别模型组成

某商店销售A、B两种商品。每件A利润20元，每件B利润15元。每天最多可使用60小时人工，A每件需要3小时，B每件需要2小时。

请写出：

1. 决策变量；
2. 目标函数；
3. 人工约束；
4. 非负约束。

## 练习2：判断可行解

给定模型：

```text
max z = 5x1 + 4x2

s.t.
    2x1 + x2 <= 10
    x1 + 2x2 <= 8
    x1, x2 >= 0
```

判断下列点是否为可行解，并说明原因：

1. `(2, 2)`
2. `(4, 2)`
3. `(-1, 3)`

## 练习3：完整建模

某车间生产桌子和椅子。每张桌子利润80元，每把椅子利润50元。每张桌子需要4小时木工和2小时油漆，每把椅子需要2小时木工和1小时油漆。每周可用木工时间120小时、油漆时间50小时。市场要求桌子最多生产20张。

建立利润最大的线性规划模型。

## 练习4：判断是否线性

判断下列式子是否线性：

1. `3x1 + 5x2`
2. `x1^2 + x2`
3. `2x1x2`
4. `4x1 - 3x2 <= 10`
5. `sqrt(x1) + x2`

## 答案

### 练习1

设 `x1` 为A商品销售量，`x2` 为B商品销售量。

```text
max z = 20x1 + 15x2

s.t.
    3x1 + 2x2 <= 60
    x1, x2 >= 0
```

### 练习2

1. `(2, 2)`：可行。
   - `2×2 + 2 = 6 <= 10`
   - `2 + 2×2 = 6 <= 8`

2. `(4, 2)`：可行。
   - `2×4 + 2 = 10 <= 10`
   - `4 + 2×2 = 8 <= 8`

3. `(-1, 3)`：不可行，因为违反 `x1 >= 0`。

### 练习3

设 `x1` 为桌子产量，`x2` 为椅子产量。

```text
max z = 80x1 + 50x2

s.t.
    4x1 + 2x2 <= 120
    2x1 + x2 <= 50
    x1 <= 20
    x1, x2 >= 0
```

### 练习4

1. 线性
2. 非线性
3. 非线性
4. 线性
5. 非线性
