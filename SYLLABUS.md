# 运筹学零基础课程路线

## 总目标

从零基础出发，在三天、每天六小时内掌握最新考试范围内的核心概念、计算方法和常见题型，并能独立完成基础建模与计算题。

## 最新考试范围

- 第1章：1.1—1.5、1.9—1.13
- 第2章：2.1—2.2
- 第4章：全部
- 第8章：8.1—8.3

详细范围以 `exam-prep/EXAM_SCOPE.md` 为准。

## 第一阶段：第1章 线性规划基础

1. 1.1 线性规划问题与建模
2. 1.2 双变量图解法
3. 1.3 标准形
4. 1.4 基本解、基本可行解与基本最优解
5. 1.5 几何特征与线性规划基本定理

## 第二阶段：第1章 单纯形法专题

1. 1.9 最优性的检验
2. 1.10 单纯形法的步骤
3. 1.11 唯一最优、无穷多最优与其他情形
4. 1.12 大M法
5. 1.13 两阶段法

## 第三阶段：第2章 对偶理论

1. 2.1 原问题与对偶问题
2. 对偶问题的写法
3. 2.2 弱对偶、强对偶与最优性
4. 互补松弛条件
5. 对偶问题的基本性质

## 第四阶段：第4章 运输问题

1. 运输问题建模
2. 运输表、基本格子集和孤立格子集
3. 西北角法
4. 最小元素法
5. 位势法与检验数
6. 闭回路调整
7. 表上作业法

## 第五阶段：第8章 图论基础

1. 8.1 图论的起源
2. 8.2 图的基本概念
3. 握手定理、子图、连通和矩阵表示
4. 8.3 树与支撑树
5. 破圈法、避圈法与最小权支撑树

## 明确不进入课程的内容

- 1.6、1.7、1.8
- 2.3、2.4、2.5、2.6
- 第3章全部
- 第5章全部
- 第6章全部
- 第7章全部
- 8.4、8.5、8.6、8.7、8.8

## 教学节奏

每节课固定包含：

1. 零基础解释
2. 一个完整例题
3. 课堂练习
4. 答案与纠错
5. 掌握度记录
