# 运输问题后续教程：位势检验与闭回路调整

## 起点：初始基本可行解

运输表：

```text
        B1  B2  B3  B4 | ai
A1      2   9   10  7  | 9
A2      1   3   4   2  | 8
A3      8   4   2   5  | 7
-------------------------
bj      5   7   6   6
```

最小元素法得到的初始方案：

```text
x12 = 6
x14 = 3
x21 = 5
x24 = 3
x32 = 1
x33 = 6
```

初始费用：

```text
Z = 6*9 + 3*7 + 5*1 + 3*2 + 1*4 + 6*2 = 102
```

## 1. 第一次位势检验

基变量满足：

```text
u_i + v_j = c_ij
```

取 `u1=0`，由基变量得到：

```text
u1 = 0
v2 = 9
v4 = 7
u2 = -5
v1 = 6
u3 = -5
v3 = 7
```

空格检验数：

```text
r_ij = u_i + v_j - c_ij
```

主要空格：

```text
r11 = 0 + 6 - 2 = 4 > 0
r22 = -5 + 9 - 3 = 1 > 0
```

因为有正检验数，所以当前方案不是最优。

按最大正检验数选：

```text
x11 进基
```

## 2. 第一次闭回路调整

进基格：

```text
(1,1)
```

闭回路：

```text
(1,1)+ -> (1,4)- -> (2,4)+ -> (2,1)- -> 回到 (1,1)
```

负号格子为：

```text
x14 = 3
x21 = 5
```

取：

```text
θ = min(3,5) = 3
```

调整：

```text
x11 = 0 + 3 = 3
x14 = 3 - 3 = 0
x24 = 3 + 3 = 6
x21 = 5 - 3 = 2
```

其他不变：

```text
x12 = 6
x32 = 1
x33 = 6
```

新方案：

```text
x11 = 3
x12 = 6
x21 = 2
x24 = 6
x32 = 1
x33 = 6
```

新费用：

```text
Z = 3*2 + 6*9 + 2*1 + 6*2 + 1*4 + 6*2 = 90
```

也可以直接用：

```text
新费用 = 旧费用 - r11*θ = 102 - 4*3 = 90
```

## 3. 第二次位势检验

对新基变量：

```text
x11, x12, x21, x24, x32, x33
```

取 `u1=0`，得：

```text
u1 = 0
v1 = 2
v2 = 9
u2 = -1
v4 = 3
u3 = -5
v3 = 7
```

空格检验数中：

```text
r22 = -1 + 9 - 3 = 5 > 0
r23 = -1 + 7 - 4 = 2 > 0
```

选最大正检验数：

```text
x22 进基
```

## 4. 第二次闭回路调整

闭回路：

```text
(2,2)+ -> (2,1)- -> (1,1)+ -> (1,2)- -> 回到 (2,2)
```

负号格子：

```text
x21 = 2
x12 = 6
```

取：

```text
θ = min(2,6) = 2
```

调整：

```text
x22 = 0 + 2 = 2
x21 = 2 - 2 = 0
x11 = 3 + 2 = 5
x12 = 6 - 2 = 4
```

其他不变：

```text
x24 = 6
x32 = 1
x33 = 6
```

新方案：

```text
x11 = 5
x12 = 4
x22 = 2
x24 = 6
x32 = 1
x33 = 6
```

新费用：

```text
Z = 90 - 5*2 = 80
```

## 5. 第三次位势检验

基变量：

```text
x11, x12, x22, x24, x32, x33
```

取 `u1=0`，得：

```text
u1 = 0
v1 = 2
v2 = 9
u2 = -6
v4 = 8
u3 = -5
v3 = 7
```

空格检验数中：

```text
r14 = 0 + 8 - 7 = 1 > 0
```

所以：

```text
x14 进基
```

## 6. 第三次闭回路调整

闭回路：

```text
(1,4)+ -> (2,4)- -> (2,2)+ -> (1,2)- -> 回到 (1,4)
```

负号格子：

```text
x24 = 6
x12 = 4
```

取：

```text
θ = min(6,4) = 4
```

调整：

```text
x14 = 0 + 4 = 4
x24 = 6 - 4 = 2
x22 = 2 + 4 = 6
x12 = 4 - 4 = 0
```

其他不变：

```text
x11 = 5
x32 = 1
x33 = 6
```

新方案：

```text
x11 = 5
x14 = 4
x22 = 6
x24 = 2
x32 = 1
x33 = 6
```

新费用：

```text
Z = 80 - 1*4 = 76
```

## 7. 最终位势检验

最终基变量：

```text
x11, x14, x22, x24, x32, x33
```

取 `u1=0`，得：

```text
u1 = 0
v1 = 2
v4 = 7
u2 = -5
v2 = 8
u3 = -4
v3 = 6
```

检查所有空格：

```text
r12 = 0 + 8 - 9 = -1
r13 = 0 + 6 - 10 = -4
r21 = -5 + 2 - 1 = -4
r23 = -5 + 6 - 4 = -3
r31 = -4 + 2 - 8 = -10
r34 = -4 + 7 - 5 = -2
```

所有空格检验数均满足：

```text
r_ij <= 0
```

所以达到最优。

## 8. 最终答案

最优运输方案：

```text
x11 = 5
x14 = 4
x22 = 6
x24 = 2
x32 = 1
x33 = 6
```

最小总运费：

```text
Zmin = 5*2 + 4*7 + 6*3 + 2*2 + 1*4 + 6*2
     = 10 + 28 + 18 + 4 + 4 + 12
     = 76
```

## 9. 考试记忆

```text
1. 先求初始方案；
2. 基变量列方程 u_i+v_j=c_ij；
3. 空格算 r_ij=u_i+v_j-c_ij；
4. 若所有 r_ij<=0，则最优；
5. 若有 r_ij>0，选最大正数进基；
6. 画闭回路，进基格标 +，沿回路 + - + - 交替；
7. θ 取负号格中最小运输量；
8. 加号格加 θ，负号格减 θ；
9. 重复检验。
```
