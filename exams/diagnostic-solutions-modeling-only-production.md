# 查漏补缺带练：建模题只写数学模型

## 题目：生产计划建模

某工厂生产甲、乙两种产品。甲产品每件需2小时加工时间和1单位原料，乙产品每件需1小时加工时间和2单位原料。每周加工时间最多100小时，原料最多80单位。甲、乙产品单位利润分别为5元和4元。市场要求甲产品至少生产10件，乙产品至少生产15件。

建立利润最大化线性规划模型。

## 1. 设变量

```text
设 x1 表示甲产品生产数量；
设 x2 表示乙产品生产数量。
```

## 2. 目标函数

利润最大：

```text
max Z = 5x1 + 4x2
```

## 3. 约束条件

加工时间最多100小时：

```text
2x1 + x2 <= 100
```

原料最多80单位：

```text
x1 + 2x2 <= 80
```

市场最低产量要求：

```text
x1 >= 10
x2 >= 15
```

非负约束：

```text
x1, x2 >= 0
```

## 4. 完整模型

```text
设 x1, x2 分别表示甲、乙产品的生产数量。

max Z = 5x1 + 4x2
s.t.
    2x1 + x2 <= 100
    x1 + 2x2 <= 80
    x1 >= 10
    x2 >= 15
    x1, x2 >= 0
```

## 5. 建模题答题格式

建模题只写：

```text
变量
目标函数
约束条件
变量取值约束
```

不写 LINGO 代码。LINGO 代码放在“编程题”中写。
