## 第二部分：线性规划图解法

### 4. 图解法求最优解

利用图解法求解：

```text
max z = 4x1 + 3x2
s.t.
    x1 <= 6
    x2 <= 5
    x1 + 2x2 <= 10
    2x1 + x2 <= 12
    x1,x2 >= 0
```

要求：写出可行域顶点，计算各顶点目标函数值，并给出最优解。

## 第三部分：线性规划建模

### 6. 生产计划建模

某工厂生产甲、乙两种产品。甲产品每件需2小时加工时间和1单位原料，乙产品每件需1小时加工时间和2单位原料。每周加工时间最多100小时，原料最多80单位。甲、乙产品单位利润分别为5元和4元。市场要求甲产品至少生产10件，乙产品至少生产15件。

建立利润最大化线性规划模型。

### 7. 配料问题建模

某饲料厂用两种原料 A、B 配制饲料。每千克 A 含蛋白质4单位、成本3元；每千克 B 含蛋白质2单位、成本2元。要求配制的饲料总量不少于50千克，蛋白质总量不少于160单位。

建立成本最小化线性规划模型。

## 第五部分：对偶问题与松紧互补

### 10. 写出对偶问题

写出下列问题的对偶问题：

```text
max z = 3x1 + 2x2 + 4x3
s.t.
    x1 + 2x2 + x3 <= 8
    2x1 + x2 + 3x3 <= 10
    x1,x2,x3 >= 0
```

### 11. 利用松紧互补求对偶最优解

设原问题为：

```text
max z = 4x1 + 3x2
s.t.
    2x1 + x2 <= 10
    x1 + 2x2 <= 8
    x1,x2 >= 0
```

已知原问题最优解为：

```text
x* = (4,2)^T
```

求对偶问题的最优解。

### 12. 参数与对偶

设线性规划问题：

```text
max z = 2x1 + x2 + 2x3
s.t.
    x1 + x2 - x3 = 4
    x1 + x2 + kx3 <= 6
    x1,x2,x3 >= 0
```

已知最优解为：

```text
x*=(5,0,1)^T
```

求 `k`，并求其对偶问题的最优解。

## 第七部分：图与网络
