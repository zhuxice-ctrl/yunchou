## 二、作图题（本题满分10分）

```text
max z = 3x1 + 4x2
s.t.
    x1 <= 5
    x2 <= 4
    2x1 + x2 <= 10
    x1, x2 >= 0
```

## 三、建模题（本题满分10分）

建立下述问题的数学模型（只建模，不求解）：

某厂生产甲、乙两种产品，生产效率均为1件/小时，每周正常可用总工时为90小时。市场预测下一周甲、乙产品需求量分别为60件和50件，单位利润分别为4元/件和3元/件。厂方要求下一周中（按优先级从高到低）：

1. 尽量避免开工不足，以充分利用现有生产能力；
2. 加班时间不超过12小时；
3. 尽量满足市场需求，且两种产品满足市场需求的重要性按其单位利润衡量；
4. 尽量不加班。

请为该厂制定最佳生产方案的目标规划模型。

## 五、应用题（本题满分10分）

某市有8个新建居民小区，拟沿道路铺设供暖管线，要求覆盖所有小区，并尽可能节省造价。道路及长度如下：

2-4: 2
3-4: 5
3-6: 6
4-5: 3
4-7: 4
5-8: 2

## 六、计算题（本题共2小题，满分30分）

### 1.（本题满分20分）

利用单纯形法求解如下线性规划问题：

```text
min z = -4x1 - 6x2
s.t.
    2x1 + x2 <= 80
    x1 + 2x2 <= 70
    x1 + x2 <= 45
    x1, x2 >= 0
```

### 2.（本题满分10分）

设线性规划问题：

```text
max z = 2x1 + x2 + 2x3
s.t.
    x1 + x2 - x3 = 4
    x1 + x2 + kx3 <= 6
    x1, x2, x3 >= 0
```

已知其最优解为：

```text
x* = (5,0,1)^T
```

求 `k` 的值及其对偶问题的最优解。

## 七、实操题（本题满分20分）

求解如下运输问题：

```text
m = 3, n = 4

c = (4,6,8,5, 7,3,4,6, 5,9,2,7)^T

d = (8,6,7, 5,4,6,6)^T
```

要求：先列运输表，求初始基本可行解，再进行最优性检验与必要修正，给出最优运输方案和最小运输费用。
