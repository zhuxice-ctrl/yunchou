# 查漏补缺带练：单纯形法 A

## 题目

按课件单纯形表方法求解：

```text
min z = -3x1 - 5x2
s.t.
    x1 <= 4
    2x2 <= 8
    3x1 + 2x2 <= 18
    x1,x2 >= 0
```

## 1. 化为标准形式

引入松弛变量 `x3,x4,x5`：

```text
x1 + x3 = 4
2x2 + x4 = 8
3x1 + 2x2 + x5 = 18
z + 3x1 + 5x2 = 0
xj >= 0
```

## 2. 初始单纯形表

```text
xB | x1  x2  x3  x4  x5 | b
---|---------------------|---
x3 | 1   0   1   0   0  | 4
x4 | 0   2   0   1   0  | 8
x5 | 3   2   0   0   1  | 18
z  | 3   5   0   0   0  | 0
```

课件规则：

```text
z 行还有正数 -> 继续转轴；
z 行全 <= 0 -> 最优。
```

## 3. 第一次转轴

`z` 行最大正数为 `5`，所以 `x2` 进基。

最小比值：

```text
8/2 = 4
18/2 = 9
```

所以 `x4` 出基，主元为 `2`。

转轴后：

```text
xB | x1  x2  x3  x4     x5 | b
---|------------------------|---
x3 | 1   0   1   0      0  | 4
x2 | 0   1   0   1/2    0  | 4
x5 | 3   0   0   -1     1  | 10
z  | 3   0   0   -5/2   0  | -20
```

## 4. 第二次转轴

`z` 行还有正数 `3`，所以 `x1` 进基。

最小比值：

```text
4/1 = 4
10/3 = 10/3
```

取较小值 `10/3`，所以 `x5` 出基，主元为 `3`。

转轴后：

```text
xB | x1  x2  x3  x4      x5    | b
---|----------------------------|-----
x3 | 0   0   1   1/3    -1/3   | 2/3
x2 | 0   1   0   1/2     0     | 4
x1 | 1   0   0   -1/3    1/3   | 10/3
z  | 0   0   0   -3/2   -1     | -30
```

## 5. 最优判断

`z` 行所有系数均 `<=0`，所以达到最优。

令非基变量：

```text
x4 = 0, x5 = 0
```

得到：

```text
x1 = 10/3
x2 = 4
x3 = 2/3
```

原变量最优解：

```text
x* = (10/3, 4)^T
```

最优值：

```text
zmin = -30
```

## 6. 考试记忆

```text
1. <= 约束加松弛变量；
2. 目标函数写成 z + 正系数变量 = 0；
3. z 行有正数就继续转轴；
4. 进基列选 z 行最大正数；
5. 出基行用最小正比值；
6. z 行全 <=0 时最优。
```
