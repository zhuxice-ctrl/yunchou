# 快速概念过关：运输问题

## 1. `a_i` 和 `b_j`

在运输问题中：

```text
a_i 表示第 i 个产地的供应量；
b_j 表示第 j 个销地的需求量。
```

运输表中，`a_i` 通常写在表的右侧，`b_j` 通常写在表的下方。

## 2. 供需平衡运输问题

若总供应量等于总需求量，则称为供需平衡运输问题。

```text
sum a_i = sum b_j
```

若二者不相等，则需要增加虚拟产地或虚拟销地，使其转化为平衡问题。

## 3. 最小元素法

最小元素法用于求运输问题的初始基本可行解。

基本思想：

```text
优先在单位运价最小的格子安排运输量。
```

每次安排量为：

```text
min(该行剩余供应量, 该列剩余需求量)
```

安排后，若某行供应用完则划去该行；若某列需求满足则划去该列。

## 4. 位势法中的 `u_i, v_j`

对当前运输方案中的基变量格，也就是有运输量的格子，列方程：

```text
u_i + v_j = c_ij
```

任选一个位势为0，例如：

```text
u_1 = 0
```

然后逐步解出其余 `u_i, v_j`。

注意：这个等式只对基变量格成立，不对空格成立。

## 5. 检验数与最优性条件

按课件公式：

```text
r_ij = u_i + v_j - c_ij
```

只对空格计算检验数。

对于最小化运输问题，最优性条件是：

```text
所有空格 r_ij <= 0
```

若存在：

```text
r_ij > 0
```

则当前运输方案不是最优，应选择对应空格入基，作闭回路修正。

## 6. 闭回路修正

若选某空格入基，则从该空格开始找闭回路，并交替标号：

```text
+ - + - ...
```

调整量：

```text
theta = 负号格中运输量的最小值
```

修正规则：

```text
正号格加 theta；
负号格减 theta；
其他格不变。
```

减为0的负号格退出基变量。
