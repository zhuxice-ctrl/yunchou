# 查漏补缺带练：配料问题建模

## 题目

某饲料厂用两种原料 A、B 配制饲料。每千克 A 含蛋白质4单位、成本3元；每千克 B 含蛋白质2单位、成本2元。要求配制的饲料总量不少于50千克，蛋白质总量不少于160单位。

建立成本最小化线性规划模型。

## 1. 设变量

```text
设 x1 表示原料 A 的使用量（千克）；
设 x2 表示原料 B 的使用量（千克）。
```

## 2. 目标函数

成本最小：

```text
min Z = 3x1 + 2x2
```

## 3. 约束条件

饲料总量不少于50千克：

```text
x1 + x2 >= 50
```

蛋白质总量不少于160单位：

```text
4x1 + 2x2 >= 160
```

非负约束：

```text
x1, x2 >= 0
```

## 4. 完整模型

```text
设 x1, x2 分别表示原料 A、B 的使用量。

min Z = 3x1 + 2x2
s.t.
    x1 + x2 >= 50
    4x1 + 2x2 >= 160
    x1, x2 >= 0
```

## 5. 建模题识别方法

看到“成本最低、费用最小、时间最短”，目标函数通常写：

```text
min Z = ...
```

看到“不少于、至少、最低要求”，约束方向通常写：

```text
>=
```

看到“不超过、最多、资源限制”，约束方向通常写：

```text
<=
```
