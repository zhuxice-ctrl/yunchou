## 各题反馈

### 一、概念题

整体可以。注意：

```text
最优解：必须是在可行解中，使目标函数达到最大值或最小值的解。
```

弱对偶写法可以是：

```text
若 x 是原问题可行解，y 是对偶问题可行解，则 c^T x <= b^T y。
```

### 三、建模题

```text
max Z = 5x1 + 7x2
s.t.
    2x1 + x2 <= 100
```

### 四、LINGO 编程题

建议直接写：

```lingo
MODEL:

MAX = 5 * X1 + 7 * X2;

2 * X1 + X2 <= 100;
X1 + 3 * X2 <= 120;

X1 >= 10;
X2 >= 20;

END
```

不要普通模型和 SETS 写法混在一起。

### 六、对偶与松紧互补

对偶模型写对了：

```text
min w = 6y1 + 12y2
s.t.
    y1 + 2y2 >= 2
    y1 + y2 >= 1
    y1 + 3y2 >= 3
    y1,y2 >= 0
```

但还需要继续做松紧互补。

已知：

```text
x* = (0,0,4)^T
```

因为 `x3>0`，所以对偶第三个约束取等号：

```text
y1 + 3y2 = 3
```

原问题第一条约束：

```text
x1 + x2 + x3 <= 6
```

代入 `x*=(0,0,4)`：

```text
4 < 6
```

第一条约束有松弛，所以：

```text
y1 = 0
```

代入：

```text
3y2 = 3
```

得到：

```text
y2 = 1
```

所以：

```text
y* = (0,1)^T
wmin = 12
zmax = 12
```
