# 查漏补缺带练：目标规划建模题

## 题目

某厂生产甲、乙两种产品，生产效率均为1件/小时，每周正常可用总工时为90小时。市场预测下一周甲、乙产品需求量分别为60件和50件，单位利润分别为4元/件和3元/件。厂方要求下一周中，按优先级从高到低：

1. 尽量避免开工不足，以充分利用现有生产能力；
2. 加班时间不超过12小时；
3. 尽量满足市场需求，且两种产品满足市场需求的重要性按其单位利润衡量；
4. 尽量不加班。

建立目标规划模型。

## 1. 决策变量

```text
设 x1 表示甲产品生产数量；
设 x2 表示乙产品生产数量。
```

因为生产效率均为1件/小时，所以总生产工时为：

```text
x1 + x2
```

## 2. 偏差变量

设各目标的正、负偏差变量为：

```text
d1-, d1+ : 正常工时90小时目标的负偏差和正偏差；
d2-, d2+ : 加班不超过12小时目标的负偏差和正偏差；
d3-, d3+ : 甲产品需求60件目标的负偏差和正偏差；
d4-, d4+ : 乙产品需求50件目标的负偏差和正偏差。
```

其中：

```text
d- 表示未达到目标；
d+ 表示超过目标。
```

## 3. 目标约束

### 目标1：充分利用正常工时90小时

```text
x1 + x2 + d1- - d1+ = 90
```

避免开工不足，就是尽量减少：

```text
d1-
```

### 目标2：加班时间不超过12小时

由于 `d1+` 表示超过90小时的部分，即加班时间。

因此：

```text
d1+ + d2- - d2+ = 12
```

加班时间不超过12小时，就是尽量减少：

```text
d2+
```

### 目标3：尽量满足市场需求

甲产品需求60件：

```text
x1 + d3- - d3+ = 60
```

乙产品需求50件：

```text
x2 + d4- - d4+ = 50
```

没有满足需求才扣罚，所以惩罚：

```text
d3-, d4-
```

又因为甲、乙单位利润分别为4、3，所以权重为4、3。

### 目标4：尽量不加班

尽量不加班，就是尽量减少加班时间：

```text
d1+
```

## 4. 目标函数

按优先级写为：

```text
min Z = P1 d1- + P2 d2+ + P3(4d3- + 3d4-) + P4 d1+
```

其中：

```text
P1 >> P2 >> P3 >> P4
```

## 5. 完整模型

```text
设 x1, x2 分别表示甲、乙产品的生产数量。

min Z = P1 d1- + P2 d2+ + P3(4d3- + 3d4-) + P4 d1+

s.t.
    x1 + x2 + d1- - d1+ = 90
    d1+ + d2- - d2+ = 12
    x1 + d3- - d3+ = 60
    x2 + d4- - d4+ = 50

    x1, x2 >= 0
    di-, di+ >= 0, i=1,2,3,4
```

## 6. 记忆方法

目标规划建模四步：

```text
1. 设决策变量；
2. 给每个目标配一对偏差变量 d-, d+；
3. 把目标写成等式；
4. 按题目优先级写 min P1 + P2 + ...。
```

偏差选择：

```text
“避免不足” -> 惩罚 d-
“不超过” -> 惩罚 d+
“尽量满足需求” -> 惩罚需求的 d-
“尽量不加班” -> 惩罚加班变量
```
