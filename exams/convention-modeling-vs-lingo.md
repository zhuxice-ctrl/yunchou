# 考试约定：建模题与 LINGO 代码题分开

根据往年试题结构：

```text
建模题：写数学模型，不写 LINGO 代码。
编程题：把模型写成 LINGO 程序。
```

## 1. 建模题应该写什么

建模题重点是：

```text
设变量
写目标函数
写约束条件
写非负约束或变量取值约束
```

例如：

```text
设 x1, x2 分别表示甲、乙产品的生产数量。

max Z = 5x1 + 4x2
s.t.
    2x1 + x2 <= 100
    x1 + 2x2 <= 80
    x1 >= 10
    x2 >= 15
    x1, x2 >= 0
```

这就是建模题答案。

## 2. LINGO 编程题应该写什么

若题目要求“编写 LINGO 程序”，才写：

```lingo
MODEL:

MAX = 5 * X1 + 4 * X2;

2 * X1 + X2 <= 100;
X1 + 2 * X2 <= 80;

X1 >= 10;
X2 >= 15;

END
```

## 3. 后续带练规则

后续训练按考试结构分开：

```text
建模题：只写数学模型。
编程题：写 LINGO 代码。
```

如果同一道题同时要求“建立模型并写 LINGO 程序”，再两部分都写。
