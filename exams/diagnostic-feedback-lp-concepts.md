# 查漏补缺反馈：线性规划概念辨析

用户答案要点：

1. 线性规划由目标函数、约束条件、决策变量组成。
2. 可行解、最优解、可行域的理解大体正确，但表述需要更规范。
3. 松弛变量用于将不等式转化为等式。
4. 最大化转最小化的原因未写清。
5. 弱对偶性写出了 `x∈K(P), y∈K(D)` 时 `c^Tx<=b^Ty` 的形式，方向正确。

## 规范答案

### 1. 线性规划模型组成

线性规划模型一般由三部分组成：

```text
决策变量、目标函数、约束条件。
```

其中目标函数和约束条件均为决策变量的线性函数。

### 2. 可行解、最优解、可行域

满足所有约束条件的解称为可行解。

所有可行解组成的集合称为可行域。

在可行域中使目标函数达到最大值或最小值的可行解称为最优解。

### 3. 松弛变量作用

对于 `<=` 约束，引入非负松弛变量，可以把不等式化为等式：

```text
ax <= b  ->  ax + xs = b, xs>=0
```

其作用是便于建立单纯形表和寻找初始基变量。

### 4. 最大化问题转化为最小化问题

按本课程课件写法，单纯形表的最优性判断规则与目标函数行的符号约定相配套。

若原问题为：

```text
max Z = c^T x
```

可改写为：

```text
min z = -c^T x
```

最后再把最优值取相反数还原为原最大化问题的最优值。

### 5. 弱对偶性

若 `x∈K(P)`，`y∈K(D)`，则：

```text
c^T x <= b^T y
```

即原问题任一可行解的目标函数值都不超过对偶问题任一可行解的目标函数值。

若二者目标函数值相等，则它们分别为原问题和对偶问题的最优解。
