# 查漏补缺带练：LINGO 编程题（资源分配）

## 题目

把下面线性规划模型写成 LINGO 程序：

```text
max Z = 6x1 + 8x2
s.t.
    2x1 + x2 <= 120
    x1 + 3x2 <= 150
    x1 >= 20
    x2 >= 10
    x1, x2 >= 0
```

## 1. LINGO 写法

```lingo
MODEL:

MAX = 6 * X1 + 8 * X2;

2 * X1 + X2 <= 120;
X1 + 3 * X2 <= 150;

X1 >= 20;
X2 >= 10;

END
```

## 2. 对照关系

数学模型：

```text
max Z = 6x1 + 8x2
```

LINGO：

```lingo
MAX = 6 * X1 + 8 * X2;
```

数学模型：

```text
2x1 + x2 <= 120
```

LINGO：

```lingo
2 * X1 + X2 <= 120;
```

## 3. 考试注意点

1. LINGO 程序必须以 `MODEL:` 开头，以 `END` 结束。
2. 每条目标函数和约束语句后面都要写分号 `;`。
3. 普通连续变量默认非负，所以 `X1>=0, X2>=0` 可以不写。
4. 若题目要求整数变量，加：

```lingo
@GIN(X1);
@GIN(X2);
```

5. 若题目要求 0-1 变量，加：

```lingo
@BIN(X1);
@BIN(X2);
```

## 4. 快速模板

最大化问题：

```lingo
MODEL:

MAX = 目标函数;

约束1;
约束2;
约束3;

END
```

最小化问题：

```lingo
MODEL:

MIN = 目标函数;

约束1;
约束2;
约束3;

END
```
