# 查漏补缺带练：资源分配建模题

## 题目

某企业准备用甲、乙两种设备加工 A、B 两种产品。加工一件 A 产品需要甲设备 2 小时、乙设备 1 小时；加工一件 B 产品需要甲设备 1 小时、乙设备 3 小时。甲设备每周最多可用 120 小时，乙设备每周最多可用 150 小时。A、B 产品单位利润分别为 6 元和 8 元。市场要求 A 产品至少生产 20 件，B 产品至少生产 10 件。

建立利润最大化线性规划模型。

## 1. 设变量

```text
设 x1 表示 A 产品的生产数量；
设 x2 表示 B 产品的生产数量。
```

## 2. 目标函数

A 产品每件利润 6 元，B 产品每件利润 8 元，所以：

```text
max Z = 6x1 + 8x2
```

## 3. 约束条件

甲设备时间约束：

```text
2x1 + x2 <= 120
```

乙设备时间约束：

```text
x1 + 3x2 <= 150
```

市场最低产量约束：

```text
x1 >= 20
x2 >= 10
```

非负约束：

```text
x1, x2 >= 0
```

## 4. 完整模型

```text
设 x1, x2 分别表示 A、B 产品的生产数量。

max Z = 6x1 + 8x2
s.t.
    2x1 + x2 <= 120
    x1 + 3x2 <= 150
    x1 >= 20
    x2 >= 10
    x1, x2 >= 0
```

## 5. 建模题检查表

普通线性规划建模题检查四件事：

```text
1. 变量有没有设清楚；
2. 目标函数方向 max/min 有没有看清；
3. 资源约束一般是 <=；
4. 需求或最低要求一般是 >=。
```
