# 运输问题：闭回路与去除孤立格

## 1. 结论

找闭回路时，确实可以用“去除孤立格”的方法。

步骤是：

```text
1. 把当前所有基变量格子标出来；
2. 再把要进基的空格也标出来；
3. 看每一行、每一列中标记格子的个数；
4. 若某一行或某一列只有一个标记格，它不可能在闭回路中，删掉；
5. 重复删除孤立格；
6. 最后剩下的格子就是闭回路。
```

## 2. 为什么要删孤立格

闭回路必须满足：

```text
每个参与闭回路的行，至少有两个回路格；
每个参与闭回路的列，至少有两个回路格。
```

因为闭回路要“横着走、竖着走”再回到起点。

如果某一行或某一列只有一个标记格，它只能进不能出，无法形成回路，所以要删掉。

## 3. 本题第一次调整示范

初始基变量：

```text
x12, x14, x21, x24, x32, x33
```

因为 `r11>0`，所以让：

```text
x11 进基
```

现在标记格子为：

```text
x11, x12, x14, x21, x24, x32, x33
```

按行看：

```text
第1行：x11, x12, x14  有3个
第2行：x21, x24       有2个
第3行：x32, x33       有2个
```

按列看：

```text
第1列：x11, x21       有2个
第2列：x12, x32       有2个
第3列：x33            只有1个 -> 删 x33
第4列：x14, x24       有2个
```

删掉 `x33` 后，再看：

```text
第3行：只剩 x32 一个 -> 删 x32
```

删掉 `x32` 后，再看：

```text
第2列：只剩 x12 一个 -> 删 x12
```

最后剩下：

```text
x11, x14, x24, x21
```

所以闭回路是：

```text
x11 -> x14 -> x24 -> x21 -> x11
```

也就是：

```text
(1,1)+ -> (1,4)- -> (2,4)+ -> (2,1)- -> 回到 (1,1)
```

## 4. 第二次调整示范

第二次进基格是：

```text
x22
```

当时基变量为：

```text
x11, x12, x21, x24, x32, x33
```

加入 `x22` 后标记：

```text
x11, x12, x21, x22, x24, x32, x33
```

按行列删孤立格：

```text
第4列只有 x24 -> 删 x24
第3列只有 x33 -> 删 x33
第3行只剩 x32 -> 删 x32
```

最后剩下：

```text
x22, x21, x11, x12
```

闭回路为：

```text
(2,2)+ -> (2,1)- -> (1,1)+ -> (1,2)- -> 回到 (2,2)
```

## 5. 第三次调整示范

第三次进基格是：

```text
x14
```

当时基变量为：

```text
x11, x12, x22, x24, x32, x33
```

加入 `x14` 后标记：

```text
x11, x12, x14, x22, x24, x32, x33
```

删除孤立格：

```text
第1列只有 x11 -> 删 x11
第3列只有 x33 -> 删 x33
第3行只剩 x32 -> 删 x32
```

最后剩下：

```text
x14, x24, x22, x12
```

闭回路为：

```text
(1,4)+ -> (2,4)- -> (2,2)+ -> (1,2)- -> 回到 (1,4)
```

## 6. 考试写法

考试中可以不详细写删孤立格的全过程，但自己草稿一定要会用。

正式答案写：

```text
选 x11 进基。
闭回路为：
(1,1)+ -> (1,4)- -> (2,4)+ -> (2,1)- -> (1,1)
θ = min{x14, x21} = min{3,5} = 3
```

## 7. 记忆

```text
闭回路中每一行、每一列都必须成双出现。
孤立格不可能在闭回路中。
```

最关键：

```text
先加入进基空格，再删孤立格。
```
