# 查漏补缺带练：建模题与 LINGO 代码

## 题目：生产计划建模

某工厂生产甲、乙两种产品。甲产品每件需2小时加工时间和1单位原料，乙产品每件需1小时加工时间和2单位原料。每周加工时间最多100小时，原料最多80单位。甲、乙产品单位利润分别为5元和4元。市场要求甲产品至少生产10件，乙产品至少生产15件。

建立利润最大化线性规划模型，并写出 LINGO 程序。

## 1. 决策变量

```text
X1 = 甲产品生产数量
X2 = 乙产品生产数量
```

## 2. 目标函数

利润最大：

```text
MAX = 5*X1 + 4*X2
```

## 3. 约束条件

加工时间约束：

```text
2*X1 + X2 <= 100
```

原料约束：

```text
X1 + 2*X2 <= 80
```

最低产量要求：

```text
X1 >= 10
X2 >= 15
```

非负约束：

```text
X1 >= 0
X2 >= 0
```

## 4. 数学模型

```text
max Z = 5X1 + 4X2
s.t.
    2X1 + X2 <= 100
    X1 + 2X2 <= 80
    X1 >= 10
    X2 >= 15
    X1, X2 >= 0
```

## 5. LINGO 程序

```lingo
MODEL:

MAX = 5 * X1 + 4 * X2;

2 * X1 + X2 <= 100;
X1 + 2 * X2 <= 80;

X1 >= 10;
X2 >= 15;

END
```

## 6. 考试写法

建模题若要求 LINGO 代码，作答结构建议为：

```text
1. 设变量；
2. 写目标；
3. 写约束；
4. 给出 LINGO 程序。
```

LINGO 中变量默认非负，一般可以不单独写 `X1>=0, X2>=0`。若题目要求整数变量，应加入：

```lingo
@GIN(X1);
@GIN(X2);
```

若题目要求 0-1 变量，应加入：

```lingo
@BIN(X1);
@BIN(X2);
```
