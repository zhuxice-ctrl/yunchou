# 查漏补缺带练：对偶问题与松紧互补 A

## 题目

已知原问题：

```text
max z = 3x1 + 2x2
s.t.
    x1 + x2 <= 4
    2x1 + x2 <= 5
    x1, x2 >= 0
```

1. 写出其对偶问题；
2. 已知原问题最优解为 `x*=(1,3)^T`，利用松紧互补求对偶最优解。

## 1. 写对偶问题

原问题是：

```text
max
<= 约束
x >= 0
```

所以对偶是：

```text
min
>= 约束
y >= 0
```

设两个约束对应的对偶变量为 `y1,y2`。

原问题右端项为 `4,5`，所以对偶目标函数为：

```text
min w = 4y1 + 5y2
```

原问题变量 `x1` 的系数列为：

```text
1, 2
```

因此对应对偶约束：

```text
y1 + 2y2 >= 3
```

原问题变量 `x2` 的系数列为：

```text
1, 1
```

因此对应对偶约束：

```text
y1 + y2 >= 2
```

所以对偶问题为：

```text
min w = 4y1 + 5y2
s.t.
    y1 + 2y2 >= 3
    y1 + y2 >= 2
    y1,y2 >= 0
```

## 2. 用松紧互补求对偶最优解

已知原问题最优解：

```text
x1 = 1, x2 = 3
```

因为：

```text
x1 > 0, x2 > 0
```

所以对偶的两个约束都取等号：

```text
y1 + 2y2 = 3
 y1 + y2 = 2
```

两式相减：

```text
y2 = 1
```

代回：

```text
y1 + 1 = 2
```

得到：

```text
y1 = 1
```

所以：

```text
y* = (1,1)^T
```

对偶最优值：

```text
wmin = 4*1 + 5*1 = 9
```

原问题最优值：

```text
zmax = 3*1 + 2*3 = 9
```

二者相等，符合强对偶定理。

## 3. 最终答案

```text
对偶问题：
min w = 4y1 + 5y2
s.t.
    y1 + 2y2 >= 3
    y1 + y2 >= 2
    y1,y2 >= 0

对偶最优解：
y* = (1,1)^T

对偶最优值：
wmin = 9
```

## 4. 考试记忆

```text
原问题 max + <= + x>=0
对应
对偶 min + >= + y>=0
```

松紧互补：

```text
xj > 0  -> 对偶第 j 个约束取等号；
yi > 0  -> 原问题第 i 个约束取等号；
原约束有松弛 -> 对应 yi = 0；
对偶约束有松弛 -> 对应 xj = 0。
```
