# 查漏补缺带练：图解法题

题目：

```text
max z = 4x1 + 3x2
s.t.
    x1 <= 6
    x2 <= 5
    x1 + 2x2 <= 10
    2x1 + x2 <= 12
    x1,x2 >= 0
```

## 1. 画约束边界

边界直线：

```text
x1=6
x2=5
x1+2x2=10
2x1+x2=12
x1=0
x2=0
```

因为全部为 `<=`，可行域取靠近原点的一侧。

## 2. 求可行域顶点

坐标轴上的顶点：

```text
(0,0)
(6,0)
(0,5)
```

两条斜线交点：

```text
x1+2x2=10
2x1+x2=12
```

解得：

```text
x2=8/3
x1=14/3
```

所以交点为：

```text
(14/3, 8/3)
```

可行域顶点：

```text
(0,0), (6,0), (14/3,8/3), (0,5)
```

## 3. 计算目标函数值

目标函数：

```text
z=4x1+3x2
```

| 顶点 | 目标函数值 |
|---|---:|
| (0,0) | 0 |
| (6,0) | 24 |
| (14/3,8/3) | 80/3 |
| (0,5) | 15 |

最大值为：

```text
80/3
```

## 4. 最优解

```text
x1=14/3
x2=8/3
zmax=80/3
```

考试作答格式：

```text
由图可知，可行域顶点为 (0,0), (6,0), (14/3,8/3), (0,5)。
分别代入目标函数 z=4x1+3x2，得 0,24,80/3,15。
故最优解为 x*=(14/3,8/3)^T，最大值 zmax=80/3。
```
