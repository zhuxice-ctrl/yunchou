# 查漏补缺带练：单纯形法（接近往年题）

## 题目

用单纯形法求解：

```text
min z = -5x1 - 4x2
s.t.
    x1 + 3x2 <= 90
    2x1 + x2 <= 80
    x1 + x2 <= 45
    x1, x2 >= 0
```

## 1. 化为标准形式

引入松弛变量 `x3,x4,x5`：

```text
x1 + 3x2 + x3 = 90
2x1 + x2 + x4 = 80
x1 + x2 + x5 = 45
z + 5x1 + 4x2 = 0
```

## 2. 初始单纯形表

```text
xB | x1  x2  x3  x4  x5 | b
---|---------------------|---
x3 | 1   3   1   0   0  | 90
x4 | 2   1   0   1   0  | 80
x5 | 1   1   0   0   1  | 45
z  | 5   4   0   0   0  | 0
```

## 3. 第一次转轴

`z` 行最大正数为 `5`，所以 `x1` 进基。

最小正比值：

```text
90/1 = 90
80/2 = 40
45/1 = 45
```

取最小值 `40`，所以 `x4` 出基。

主元为 `2`。

转轴后：

```text
xB | x1  x2    x3  x4      x5 | b
---|---------------------------|---
x3 | 0   5/2   1   -1/2    0  | 50
x1 | 1   1/2   0    1/2    0  | 40
x5 | 0   1/2   0   -1/2    1  | 5
z  | 0   3/2   0   -5/2    0  | -200
```

## 4. 第二次转轴

`z` 行还有正数 `3/2`，所以 `x2` 进基。

最小正比值：

```text
50/(5/2) = 20
40/(1/2) = 80
5/(1/2) = 10
```

取最小值 `10`，所以 `x5` 出基。

主元为 `1/2`。

转轴后：

```text
xB | x1  x2  x3  x4   x5 | b
---|----------------------|---
x3 | 0   0   1   2    -5 | 25
x1 | 1   0   0   1    -1 | 35
x2 | 0   1   0  -1     2 | 10
z  | 0   0   0  -1    -3 | -215
```

## 5. 最优判断

`z` 行所有检验数均 `<=0`，所以达到最优。

令非基变量：

```text
x4 = 0, x5 = 0
```

从表中读出：

```text
x1 = 35
x2 = 10
x3 = 25
```

所以原变量最优解为：

```text
x* = (35, 10)^T
```

最优值为：

```text
zmin = -215
```

## 6. 最终答案

```text
x1 = 35, x2 = 10
zmin = -215
```
